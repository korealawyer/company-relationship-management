# 🧠 논리구조 & 장표 아이디어 뱅크

## 🎯 이 뱅크의 목적

**"메일·제안서 만들 때 여기서 뽑아쓴다"**

- 검증된 논리구조 패턴 모음
- 와디즈식 장표 템플릿
- 상황별 조합 가이드
- 효과성 점수 추적 (⭐~⭐⭐⭐⭐⭐)

---

## 📊 논리구조 패턴 라이브러리

### Pattern 1: 문제-비용-해결 (PCS)

**언제**: 1차 메일, 초기 제안서

```
[Problem] 프랜차이즈 본부가 겪는 3가지 법률 리스크
  1. 가맹점 분쟁 (평균 해결 비용 500만원)
  2. 정보공개서 미비 (과태료 최대 3,000만원)
  3. 공정거래위원회 제재 (브랜드 이미지 타격)

[Cost] 방치했을 때 실제 비용
  • 연간 분쟁 2건 발생 시: 1,000만원
  • 변호사 시간당 비용: 30~50만원
  • 기회비용: 대표님 시간 낭비

[Solution] IBS 월 100만원으로 모든 리스크 제로화
  • 무제한 법률상담 (본사+가맹점+임직원)
  • 예방 중심 시스템 (분쟁 전 차단)
  • 법률문서 2,000종 제공
```

**효과성**: ⭐⭐⭐⭐⭐ (논리적 설득에 최적)

---

### Pattern 2: ROI 증명 (Investment-Return)

**언제**: 2차 메일, 가격 부담 우려 시

```
[Investment] 30만원 (고객 1개사 유치 비용)
  ↓
[Process] 9일 만에 투자금 회수
  ↓
[Return] 3년간 3,600만원 (LTV)
  ↓
[Ratio] ROI 120배

🎯 결론: "첫 달부터 대폭 순이익"
```

**장표**: 

```
━━━━━━━━━━━━━━━━━━━━━━━
   30만원  →  3,600만원
   투자        회수 (3년)
   
   💰 ROI 120배
   📅 회수 기간 9일
━━━━━━━━━━━━━━━━━━━━━━━
```

**효과성**: ⭐⭐⭐⭐⭐ (숫자 중심 설득)

---

### Pattern 3: Before/After 대비

**언제**: 3차 메일, 제안서 핵심 섹션

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
         BEFORE              AFTER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
법률 분쟁 발생 시        │  사전 예방으로
건당 500만원            │  분쟁 제로
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
변호사 찾기             │  전담 변호사
2주 소요                │  즉시 연결
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
가맹점 CS 부담          │  법률상담 제공
월 20시간               │  자동 분산
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 결론: 분쟁 1건만 막아도 5개월치 무료
```

**효과성**: ⭐⭐⭐⭐ (시각적 임팩트 강함)

---

### Pattern 4: 3단계 여정 (Journey)

**언제**: 온보딩 설명, 장기 제안

```
STEP 1: 현재 상태 진단 (1주)
  ✓ 계약서 검토
  ✓ 정보공개서 점검
  ✓ 리스크 리포트 제공
  
STEP 2: 시스템 구축 (1개월)
  ✓ 법률상담 채널 오픈
  ✓ 가맹점 지원 프로그램 세팅
  ✓ 표준 문서 정비
  
STEP 3: 안정 운영 (지속)
  ✓ 분기별 리스크 브리핑
  ✓ 정책 업데이트
  ✓ 성장 단계별 가이드
```

**효과성**: ⭐⭐⭐⭐ (프로세스 신뢰 구축)

---

### Pattern 5: 사회적 증거 (Social Proof)

**언제**: 3차 메일, 신뢰 구축 필요 시

```
🏆 초기 고객 성과

"계약서 검토만으로도 월 비용 뽑았어요"
- 치킨 프랜차이즈 A 대표 (가맹점 15개)
- 계약 후 1개월, 가맹점 분쟁 1건 사전 차단 → 500만원 절약

"CS 부담이 50% 줄었습니다"
- 카페 B 본부 (가맹점 80개)
- 가맹점 법률상담 프로그램 도입 후 본사 CS 문의 절반 감소

"이제 법률은 걱정 안 합니다"
- 교육 프랜차이즈 C 대표 (가맹점 30개)
- 전담 변호사 배정으로 의사결정 속도 3배 향상
```

**효과성**: ⭐⭐⭐⭐⭐ (후기는 최강 설득 도구)

## 🎨 와디즈식 장표 템플릿

### Template 1: 숫자 강조 박스

```
┌─────────────────────────────────┐
│                                 │
│       30만원 → 3,600만원        │
│                                 │
│         💰 ROI 120배            │
│         📅 회수 9일             │
│                                 │
└─────────────────────────────────┘
```

**사용**: ROI, 성장 지표, 절감 효과

---

### Template 2: 3분할 비교표

```
┌──────────┬──────────┬──────────┐
│  경쟁사   │  대형로펌  │   IBS    │
├──────────┼──────────┼──────────┤
│ 월 100만원│ 월 300만원│ 월 100만원│
│ 템플릿30개│ 시간당 과금│ 무제한 상담│
│ 일반 법률 │ 일반 법률 │ 프랜차이즈│
│          │          │  특화    │
└──────────┴──────────┴──────────┘

✅ IBS = 가맹사업법 전문가
```

**사용**: 경쟁 비교, 차별점 강조

---

### Template 3: 타임라인 (성장 로드맵)

```
2026.02 ━━━━━━→ 2026.07 ━━━━━━→ 2026.12
  20개사          500개사         1,000개사
   ↓               ↓               ↓
 제품 검증       채널 확장       시장 1위
```

**사용**: 사업 계획, 성장 스토리

---

### Template 4: 리스크 경고 박스

```
⚠️ 방치 시 예상 비용
━━━━━━━━━━━━━━━━━━━━━━
가맹점 분쟁 2건/년    1,000만원
정보공개서 과태료      300만원
변호사 자문 10시간     500만원
━━━━━━━━━━━━━━━━━━━━━━
합계                 1,800만원

✅ IBS 연간 비용: 1,200만원
   → 연 600만원 절약
```

**사용**: 긴급성 강조, 위기 인식

---

### Template 5: 단계별 체크리스트

```
✅ 1단계: 현재 상태 진단
  ├─ 계약서 검토
  ├─ 정보공개서 점검
  └─ 리스크 리포트
  
✅ 2단계: 시스템 구축
  ├─ 상담 채널 오픈
  ├─ 문서 표준화
  └─ 교육 자료 제공
  
🔄 3단계: 지속 운영
  ├─ 분기별 브리핑
  ├─ 정책 업데이트
  └─ 성장 지원
```

**사용**: 프로세스 설명, 온보딩 가이드

## 🎯 상황별 조합 가이드

### 상황 1: 초기 콜드메일 (1차)

**조합**: Pattern 1 (문제-비용-해결) + Template 4 (리스크 경고)

### 상황 2: ROI 설득 메일 (2차)

**조합**: Pattern 2 (ROI 증명) + Template 1 (숫자 강조 박스) + Template 3 (타임라인)

### 상황 3: 신뢰 구축 메일 (3차)

**조합**: Pattern 5 (사회적 증거) + Template 3 (Before/After)

### 상황 4: 제안서 작성

**조합**: Pattern 1 + 2 + 3 전체 + 모든 Template 활용

### 상황 5: 가격 부담 우려 대응

**조합**: Pattern 2 (ROI) + Template 2 (비교표) + "분쟁 1건만 막아도 5개월치 무료"
