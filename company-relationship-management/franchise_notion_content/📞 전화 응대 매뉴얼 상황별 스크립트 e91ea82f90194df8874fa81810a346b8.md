# 📞 전화 응대 매뉴얼 | 상황별 스크립트

## 🎯 전화 응대의 원칙

**"첫 30초가 미팅을 결정한다"**

- 사전 메일 발송 필수 (Cold Call 금지)
- 상대방 상황 먼저 확인
- 3분 안에 핵심 가치 전달
- 다음 액션 명확히 제시

---

## 📋 상황별 스크립트

### 상황 1: Hot Lead 전화 (메일 응답 후)

**목표**: 미팅 일정 확정

#### 오프닝 (30초)

```
안녕하세요, 회사명 대표님.
IBS 프랜차이즈 법률자문팀 이범휘입니다.

어제(오늘 오전) 보내드린 메일에 
답장(관심 표시) 주셔서 감사합니다.

지금 통화 괜찮으신가요? 
3분만 시간 내주실 수 있을까요?
```

**⏸️ 대기**: 상대방 답변

---

#### 본론 1: 훅 리마인드 (1분)

```
메일에서 말씀드렸듯이, 
귀사 개인정보처리방침에서 
3가지 시정 필요 사항을 발견했습니다.

특히 [구체적 항목 1개]는 
과태료 위험이 있어서 
빠른 조치가 필요합니다.

혹시 이 부분 확인해보셨나요?
```

**⏸️ 대기**: 대표님 반응

---

#### 본론 2: 가치 제안 (1분)

**패턴 A**: 관심 있는 경우

```
저희 IBS는 프랜차이즈 법률 전문팀입니다.

월 100만원으로 
▸ 본사 운영 법무 무제한 상담
▸ 가맹점 법률상담 프로그램
▸ 법률문서 2,000종 제공

모두 포함됩니다.

실제로 치킨 프랜차이즈 A사는 
계약서 검토만으로 
가맹점 분쟁 1건을 사전에 차단해서 
500만원을 절약했습니다.
```

**패턴 B**: 가격 부담 우려 시

```
소규모 본부를 위한 
맞춤 플랜도 있습니다.

가맹점 가맹점수개 규모면 
월 금액만원부터 시작 가능합니다.

분쟁 1건만 막아도 
5개월치가 무료인 셈입니다.
```

---

#### 클로징: 미팅 제안 (30초)

```
자세한 내용은 
30분 미팅으로 설명드리겠습니다.

▸ 귀사 리스크 진단 결과
▸ 맞춤형 해결 방안
▸ 실제 적용 사례

모두 보여드리겠습니다.

이번 주 중 언제가 편하신가요?
화요일 오전 / 수요일 오후 중에요?
```

**🎯 목표**: 구체적 일정 확정

---

### 상황 2: Warm Lead 전화 (메일 오픈, 미응답)

**목표**: 관심 확인 → 미팅 또는 추가 자료 발송

#### 오프닝 (30초)

```
안녕하세요, 회사명 대표님.
IBS 프랜차이즈 법률자문팀 이범휘입니다.

지난 주 개인정보처리방침 관련 메일 
보내드렸는데 확인하셨나요?

지금 통화 괜찮으실까요?
```

**⏸️ 대기**: 반응 확인

---

#### 본론: 부드러운 확인 (2분)

**패턴 A**: "못 봤어요"

```
괜찮습니다! 
간단히 말씀드리면,

프랜차이즈 본부가 흔히 놓치는 
법률 리스크 3가지를 
저희가 무료로 점검해드리고 있습니다.

특히 회사명은 업종 업종이시니
▸ 가맹계약서 
▸ 정보공개서 
▸ 개인정보 처리

이 세 가지가 중요합니다.

무료 체크리스트 
카톡이나 메일로 보내드릴까요?
```

**패턴 B**: "봤는데 바빠서"

```
바쁘신 거 이해합니다.

그래서 15분 통화로 
핵심만 빠르게 전달드립니다.

▸ 현재 리스크 진단
▸ 우선 조치 사항
▸ 월 100만원 솔루션 소개

이번 주 중 
15분만 시간 내주실 수 있을까요?
```

---

### 상황 3: 가격 문의 전화

**목표**: 가격 부담 완화 → 가치 강조 → ROI 증명

```
가격 문의 주셔서 감사합니다.

기본 플랜은 월 100만원입니다만,
회사명 규모(가맹점 가맹점수개)에 맞춰
맞춤 견적을 드릴 수 있습니다.

━━━━━━━━━━━━━━━━━━━━━━━
💰 소규모 본부 맞춤 가격
━━━━━━━━━━━━━━━━━━━━━━━

가맹점 10개 미만: 월 60만원
가맹점 10~20개: 월 80만원  
가맹점 20~30개: 월 100만원

━━━━━━━━━━━━━━━━━━━━━━━

그런데 대표님,
이렇게 생각해보시면 어떨까요?

가맹점 분쟁 1건 발생하면 
평균 해결 비용이 500만원입니다.

저희 서비스로 
1년에 분쟁 1건만 막아도 
연간 비용(100만원×12=1,200만원)보다 
300만원이 남습니다.

게다가 
▸ 대표님 시간 절약
▸ 가맹점 신뢰 유지  
▸ 브랜드 리스크 제로

이런 효과는 돈으로 환산 불가능합니다.

혹시 바우처 보유하고 계신가요?
중소벤처부 법률자문 바우처 있으시면
월 50만원 실부담으로 가능합니다.
```

### 상황 4: 거절 대응

#### 거절 유형 A: "지금은 필요 없어요"

```
이해합니다.

그럼 이렇게 해드릴게요.

▸ 무료 법률 체크리스트 발송
▸ 3개월 후 재연락

괜찮으실까요?

실제로 많은 본부들이 
"필요 없다"고 하시다가 
분쟁 터진 후에 연락 주십니다.

그때는 이미 늦은 경우가 많아서...

일단 체크리스트만이라도 
받아두시면 
자체 점검에 도움 되실 겁니다.

메일 주소 알려주시겠어요?
```

---

#### 거절 유형 B: "비용이 부담돼요"

```
충분히 이해합니다.

그런데 대표님,
이건 비용이 아니라 **투자**입니다.

━━━━━━━━━━━━━━━━━━━━━
투자 vs 손실
━━━━━━━━━━━━━━━━━━━━━

투자: 월 100만원 (연 1,200만원)

방치 시 손실:
  ▸ 분쟁 1건: 500만원
  ▸ 과태료: 300만원
  ▸ 변호사 비용: 500만원
  ──────────────────
  합계: 1,300만원

━━━━━━━━━━━━━━━━━━━━━

그래도 부담되시면
▸ 소규모 플랜 (월 60만원)
▸ 바우처 활용 (월 50만원)
▸ 첫 달 50% 할인

이런 옵션도 있습니다.

일단 미팅 한 번 하시고 
결정하시는 게 어떨까요?
```

---

#### 거절 유형 C: "다른 데랑 비교해볼게요"

```
좋습니다! 
당연히 비교 검토하셔야죠.

비교하실 때 
이 3가지만 체크해보세요:

1️⃣ 가맹사업법 전문성
   → 일반 법률 vs 프랜차이즈 특화

2️⃣ 가맹점 지원 포함 여부  
   → 본사만 vs 본사+가맹점

3️⃣ 실제 비용
   → 시간당 과금 vs 무제한

저희는 
✅ 가맹사업법 10년 전문가
✅ 가맹점 상담 프로그램 포함
✅ 무제한 상담 (월 고정가)

비교해보시고 
궁금하신 점 있으면 
언제든 연락주세요.

제 번호 남겨드릴게요.
02-XXXX-XXXX / 이범휘입니다.
```

## 🎯 통화 후 액션

### ✅ 즉시 처리 (10분 내)

1. **CRM 업데이트**
    - 통화 내용 요약
    - 관심도 레벨 (Hot/Warm/Cold)
    - 다음 액션 일정
2. **팔로업 메일 발송**

```
대표님, 방금 통화 감사했습니다.

말씀드린 내용 정리해서 보내드립니다:
▸ [통화 내용 요약]
▸ [약속한 자료 첨부]
▸ [다음 일정]

궁금하신 점 있으면 언제든 연락주세요.
```

1. **Calendly 초대 (미팅 확정 안 된 경우)**

| 거절 (Hard) | 6개월 후 재연락 설정 | 180일 후 |

## 🚫 절대 금지 사항

❌ **Cold Call 금지**: 사전 메일 없이 전화 불가

❌ **30초 이상 떠들기**: 상대방 말 끊지 않기

❌ **가격부터 언급**: 가치 먼저, 가격은 나중에

❌ **강압적 영업**: "지금 결정하세요" 금지

❌ **경쟁사 비방**: 우리 강점만 강조
