# 💬 마케팅 멘트 아이디어 뱅크

## 🎯 이 뱅크의 목적

**"강력한 한 줄을 찾을 때 여기서 뽑는다"**

- 검증된 카피라이팅 패턴
- 감정 자극 멘트
- 제목/오프닝/CTA 모음
- A/B 테스트 결과 반영

## 📧 이메일 제목 라이브러리

### 카테고리 1: 훅 중심 (1차 메일용)

**패턴**: [회사명] + 구체적 발견 + 숫자

```
⭐⭐⭐⭐⭐ [회사명님] 귀사 개인정보처리방침에서 3가지 법률 리스크 발견
⭐⭐⭐⭐ [긴급] 회사명 정보공개서에 과태료 위험 조항 발견
⭐⭐⭐⭐ 회사명 가맹계약서, 공정위 제재 받을 수 있습니다
⭐⭐⭐ 귀사 홈페이지 법률 검토 결과 - 무료 리포트 첨부
```

**효과**: 개인화 + 구체성 → 오픈율 40%+

---

### 카테고리 2: 숫자 임팩트 (2차 메일용)

**패턴**: 투자 금액 → 회수 금액 + 배수

```
⭐⭐⭐⭐⭐ 월 100만원 투자 → 3년 3,600만원 회수 (ROI 120배)
⭐⭐⭐⭐ 30만원으로 시작해서 3,600만원 버는 법
⭐⭐⭐⭐ 9일 만에 투자금 회수 | 프랜차이즈 법률 자동화
⭐⭐⭐ 첫 달부터 순이익 | CAC 회수 기간 단 9일
```

**효과**: ROI 강조 → 관심 고객 필터링

---

### 카테고리 3: 긴급성/희소성 (3차 메일용)

**패턴**: [마감 표시] + 한정 수량 + 혜택

```
⭐⭐⭐⭐⭐ [조기 마감 임박] 2월 한정 3개사만 - 첫 달 50% 할인
⭐⭐⭐⭐ [D-3] 회사명님, 2월 특별 혜택 3일 남았습니다
⭐⭐⭐⭐ [1자리 남음] 프랜차이즈 법률 파트너 2월 모집 마감
⭐⭐⭐ 내일 마감 | 가맹계약서 무상 세팅 (88만원 상당)
```

**효과**: FOMO 자극 → 즉시 행동 유도

---

### 카테고리 4: 질문형 (관심 유도)

**패턴**: 고객 고민 질문 + 해결책 암시

```
⭐⭐⭐⭐ 회사명님, 가맹점 분쟁 대응에 연간 얼마 쓰고 계신가요?
⭐⭐⭐⭐ 정보공개서 등록, 혼자 하시나요? (과태료 위험)
⭐⭐⭐ 가맹점주 법률 문의, 어떻게 처리하고 계신가요?
⭐⭐⭐ 프랜차이즈 본부에 전담 변호사가 필요한 이유
```

**효과**: 공감 형성 → 열람 후 고민 자극

## 🎤 오프닝 멘트 라이브러리

### 오프닝 1: 공감 + 문제 제기

```
안녕하세요, 회사명 대표님.
저희는 프랜차이즈 법률 전문 IBS 팀입니다.

프랜차이즈 본부를 운영하시면서
이런 고민 해보신 적 있으신가요?

• 가맹점 분쟁이 터지면 어디서부터 손대야 할지 막막하다
• 정보공개서 업데이트를 계속 미루고 있다
• 변호사 자문이 필요한데 비용이 부담스럽다

이 고민, 저희가 월 100만원으로 해결해드립니다.
```

**언제**: 일반적인 1차 메일

---

### 오프닝 2: 강력한 훅 (개인정보보호정책)

```
회사명 대표님, 중요한 사실을 발견했습니다.

귀사 홈페이지의 개인정보처리방침을 검토한 결과,
**3가지 시정 필요 사항**이 확인되었습니다:

1. 제3자 제공 동의 절차 누락 → 과태료 위험
2. 개인정보 보유기간 명시 미흡 → 개인정보보호법 위반
3. 가맹점 고객정보 처리 근거 불명확 → 집단 소송 리스크

이런 이슈는 방치하면 큰 문제가 됩니다.
무료 30분 컨설팅으로 정확한 해결책을 알려드립니다.
```

**언제**: 개인정보보호정책 분석 훅 사용 시

---

### 오프닝 3: 숫자로 시작

```
30만원을 투자하면 3,600만원을 벌 수 있다면?

그것도 9일 만에 투자금을 회수하고,
첫 달부터 대폭 순이익을 낼 수 있다면?

회사명 대표님,
이것은 IBS 프랜차이즈 법률자문의 실제 ROI입니다.
```

**언제**: 2차 메일 (ROI 강조)

---

### 오프닝 4: 사회적 증거

```
이번 달 이미 2개 프랜차이즈 본부가
IBS 법률자문을 선택했습니다.

"계약서 검토만으로도 월 비용 뽑았어요"
- 치킨 프랜차이즈 A 대표 (가맹점 15개)

"CS 부담이 50% 줄었습니다"
- 카페 B 본부 (가맹점 80개)

회사명님도 이 대열에 합류하시겠습니까?
```

**언제**: 3차 메일 (신뢰 구축)
