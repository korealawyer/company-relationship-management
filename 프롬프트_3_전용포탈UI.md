# [AI #3 전용] IBS CRM 의뢰인 전용 포털 UI 구축 프롬프트

당신은 IBS 로펌 CRM 소속 프론트엔드 리드 개발자입니다. 다음의 프론트엔드 UI 컴포넌트 2가지를 각각 작성해 주세요. 코드는 바로 복붙해서 실행 가능하도록 풀 코드로 제공해 주세요.

## 🎯 과제 1: `PersonalLitigationDashboard.tsx` 딥링크 & 모달 보완
**요구사항**:
- 위치: `src/app/personal-litigation/PersonalLitigationDashboard.tsx`
- **UI 수정 1**: 우측 `DetailPanel`의 담당변호사 섹션 밑에 `📱 의뢰인 안심 포털 구축` 박스를 추가하고 `포털 초대장 발송` 기능을 시각적으로 배치하세요. (클릭 시 간단한 브라우저 알림/토스트)
- **모달 수정 2**: `AddCaseModal` 내 기일 추가 부분에서 '변론기일', '준비기일'을 선택하는 Select Box와 불변기일을 표시하는 체크박스(🔴 빨간색 텍스트 지원) UI 코드를 적용하세요. 

## 🎯 과제 2: 의뢰인 전용 포털 신규 페이지 생성
**요구사항**: 
- 위치: `src/app/client-portal/my-cases/page.tsx` (파일이 없으면 생성한다고 가정하고 전체 파일 작성)
- 의뢰인이 모바일이나 웹으로 접속했을 때 보는 심플하고 예쁜 진행 현황판뷰를 작성하세요.
- **필수 스펙**: 
  1. 테마 색상은 인디고/퍼플(`bg-indigo-600`, `text-indigo-600` 계열).
  2. 상단에 변호사 이름과 사건 번호 표시.
  3. "소장접수 - 1차변론 - 판결" 단계를 시각적으로 보여주는 좌우 가로형 Journey Map 컴포넌트 추가.
  4. 다가오는 최우선 법원 기일(날짜, 기일명)과 "길찾기" 모의 버튼 배치.
  5. `personalStore`를 불러와서 진행중인 첫 번째 사건을 `useState`로 띄워 데모가 가능해야 합니다.
