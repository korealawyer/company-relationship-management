# 5. 🔐 인증 시스템 (JWT RS256 + RT Rotation)

```
═══ 신규 로그인 (Green) ═══
사용자 → Green Next.js → bcrypt 검증 → JWT RS256 발급
  └─ AT: 2시간 (httpOnly Cookie)
  └─ RT: 7일 (httpOnly Cookie + Redis 저장)

═══ AT 만료 → RT Rotation ═══
사용자 → RT로 갱신 → Redis에서 RT 확인 → 새 AT+RT 발급
  └─ Grace Period: 5초 (동일 IP 제한)
  └─ 다른 IP에서 재사용 → 즉시 거부 + 알람 (Sentry→PagerDuty→Slack)

═══ 레거시 로그인 (migration_rw 계정) ═══
① bcrypt 검증 실패
② ionCube 해시 검증 (migration_rw 계정)
③ bcrypt 업그레이드 + password_legacy = NULL
④ Phase 3 + 90일 → password_legacy DROP + migration_rw DROP

═══ Blue PHP JWT 검증 ═══
Blue → JWKS 가져오기 (/.well-known/jwks.json) → Public Key로 검증
  └─ Redis 불필요 (보안 우수)
```
