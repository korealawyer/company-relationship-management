# 🪝 훅(Hook) 전략 플레이북: 개인정보·정보공개서·피자헛 기준변경 체크 v1.0

<aside>
🧠

목표: ‘법률 문서에서 발견한 포인트’로 상대가 **문제를 인지**하고, 담당자/대표가 **대화에 남아있게** 만드는 훅을 표준화합니다.

</aside>

### 1) 훅의 3단 구조(전화/메일 공용)

1. 관찰: “처리방침/정보공개서에서 __가 보였다”
2. 의미: “최근 기준에서는 __ 리스크/오해 소지가 될 수 있다”
3. 해결: “문장 1~2줄/항목 정리로 빠르게 예방 가능하다”

### 2) 훅 A — 개인정보 처리방침 기반(우선)

**잘 먹히는 프레임**

- “클레임이 아니라 예방 알림”
- “요즘 이 항목이 민원 포인트가 되는 케이스가 많다”

**체크 포인트(예시 범주)**

- 위탁/수탁 표기
- 보유·이용기간 표기
- 제3자 제공 표기
- 국외이전 관련 표기
- 개인정보 보호책임자/연락처 표기

### 3) 훅 B — 정보공개서 기반(프랜차이즈 Only)

- 모든 프랜차이즈는 공정위에 정보공개서를 제출
- 정보공개서 검토로 **활용 가능한 법률 이슈**를 발굴 가능

**체크 포인트(예시 범주)**

- 가맹금/교육비/광고비 관련 표기
- 영업지역/상권 관련 문구
- 계약기간/갱신/해지 관련 표기
- 정보 변경 공시/정기 변경 이슈

### 4) 피자헛 소송 이후 ‘작성 기준 변경’ 확인 체크리스트

- 확인 질문(리서치용)
    - 어떤 항목에서 기준이 바뀌었는가?
    - 변경 시점(대략 4월경?)과 근거는 무엇인가?
    - 실무적으로 “문구가 어떻게 바뀌는가?”
- 산출물
    - ‘변경 전/후’ 비교 1페이지
    - 전화용 훅 문장 2개

### 5) 업계/조직 현실 반영(회의 결론)

- 소규모/중견 프랜차이즈는 **법무팀이 없을 가능성**이 높다.
- 사내 변호사가 있어도 가맹점(별도 법인/개인사업자) 자문 제공이 어려울 수 있다.

→ 그래서 “법무팀 연결”을 전제로 하지 말고 **본부/총무/마케팅/외주** 루트를 기본으로 설계한다.
