## 🎯 **왜 긴급 대응이 중요한가?**

### **고객 이탈 3대 원인**

1. **느린 응답**: "문의했는데 2일째 답이 없어요"
2. **형식적 대응**: "매뉴얼만 보내주고 끝이에요"
3. **위기 시 부재**: "진짜 급할 때 연락이 안 돼요"

### **긴급 대응의 효과**

- ✅ **고객 만족도 30% 상승** (업계 연구)
- ✅ **갱신율 95% → 98%** (3%p 상승)
- ✅ **추천 확률 2배 증가**
- ✅ **입소문 효과 극대화**

## 🚨 **5가지 긴급 대응 시나리오**

### **시나리오 1: 가맹점주 소송 위협 (최고 긴급)**

**상황**:

- 월요일 오전 9시
- A 프랜차이즈 대표가 전화
- "가맹점주가 권리금 5천만원 돌려달라고 소송 예고했어요!"
- "변호사 선임했다는데 우리는 어떻게 해야 하나요?"

**❌ 나쁜 대응** (일반 로펌 스타일):

```
09:00 - 고객 전화
09:05 - "담당 변호사에게 전달하겠습니다"
10:00 - 담당 변호사 확인 (다른 미팅 중)
14:00 - 고객에게 회신: "계약서 보내주세요"
16:00 - 고객이 계약서 발송
익일 10:00 - 변호사 검토 시작
익일 17:00 - 고객에게 회신
→ **고객 불안감 최고조, 신뢰도 하락**
```

**✅ IBS 긴급 대응** (차별화):

```
09:00 - 고객 전화 접수
09:02 - CS가 즉시 담당 변호사에게 긴급 알림 (Slack)
09:05 - 담당 변호사가 고객에게 직접 전화
         "지금 바로 통화 가능하십니까? 10분 드리겠습니다"
09:15 - 전화 상담 완료
         - 계약서 즉시 전송 요청
         - 초기 대응 방향 안내 (내용증명 발송 등)
         - 오후 3시 추가 통화 약속
10:00 - 고객이 계약서 발송
11:00 - 변호사 검토 완료
11:30 - 고객에게 중간 보고 문자
         "검토 중입니다. 오후 3시 통화 때 상세히 말씀드리겠습니다"
15:00 - 2차 통화
         - 상세 법률 의견서 공유 (5페이지)
         - 3가지 대응 시나리오 제시
         - 권장 전략 안내
15:30 - 고객 결정 지원
16:00 - 내용증명 초안 작성 착수
익일 10:00 - 내용증명 최종본 전달
→ **고객: "와, 이 정도로 케어해주는구나. 평생 고객 할게요"**
```

**결과**:

- 고객 만족도: ⭐⭐⭐⭐⭐
- 고객 반응: "다른 로펌 3곳 알아봤는데 IBS가 최고예요"
- 추천 확률: 90% (해당 고객이 2개사 추천)

### **시나리오 2: 공정위 시정조치 통보 (긴급)**

**상황**:

- 목요일 오후 5시
- B 프랜차이즈에 공정위 시정조치 통보
- "정보공개서 허위 기재 혐의"
- 고객사 패닉 상태

**✅ IBS 긴급 대응**:

```
17:00 - 고객 긴급 문의 (카카오톡)
17:03 - CS가 담당 변호사에게 긴급 알림
17:10 - 담당 변호사 즉시 전화
         "지금 문서 확인 중입니다. 30분 후 다시 전화드리겠습니다"
17:40 - 1차 통화
         - 시정조치 내용 분석
         - 대응 기한 확인
         - 긴급도 평가
18:00 - 고객에게 문자
         "내일 오전 10시까지 대응 방안 드리겠습니다. 걱정 마세요"
익일 09:00 - 대표 변호사 직접 전화
         "제가 직접 챙기겠습니다"
10:00 - 대응 전략 보고서 전달 (10페이지)
         - 시정조치 분석
         - 대응 로드맵 (3주 일정)
         - 필요 자료 리스트
11:00 - 고객사 방문 미팅 (대표 변호사 직접)
14:00 - 답변서 초안 작성 착수
→ **고객: "대표 변호사가 직접 오셨어요. 진짜 믿을 만하네요"**
```

**결과**:

- 위기 → 기회로 전환
- 고객 신뢰도 급상승
- 계약 연장 즉시 결정 (2년 → 3년)

### **시나리오 3: 언론 보도 위기 (최고 긴급)**

**상황**:

- 금요일 밤 10시
- C 프랜차이즈 관련 부정적 기사 게재
- "가맹점주 집단 반발"
- 대표가 긴급 연락

**✅ IBS 긴급 대응**:

```
22:00 - 고객 긴급 전화 (대표 개인 번호로)
22:05 - 대표 변호사 직접 응대
         "지금 기사 확인했습니다. 30분 드리겠습니다"
22:35 - 긴급 전화 회의
         - 기사 내용 분석
         - 법적 쟁점 검토
         - 언론 대응 방향 조언
23:00 - 긴급 대응 가이드 문자 발송
         "1. 오늘 밤 추가 댓글 금지
          2. 내일 오전 9시 재통화
          3. 월요일 기자 브리핑 준비"
익일 09:00 - 주말 긴급 미팅
         - 대표 변호사 + 수석 변호사 투입
         - 3시간 집중 전략 회의
12:00 - 보도자료 초안 완성
14:00 - 법률 검토 의견서 완성
16:00 - 고객사 최종 승인
월요일 10:00 - 기자 브리핑 동행
→ **고객: "주말에 이렇게까지 해주시다니... 감동했습니다"**
```

**결과**:

- 위기 성공적 관리
- 언론 보도 방향 전환
- 고객사가 3개사 추천
- 추가 계약 (위기관리 패키지)

### **시나리오 4: 가맹점 집단 민원 (긴급)**

**상황**:

- 화요일 오후 2시
- D 프랜차이즈 가맹점 20개점이 연합
- "본사 정책 부당하다" 집단 항의
- 카카오톡 단톡방 폭발 중

**✅ IBS 긴급 대응**:

```
14:00 - 고객 긴급 문의
14:05 - CS가 즉시 담당 변호사 + 대표에게 알림
14:15 - 대표 변호사 직접 전화
         "지금 무슨 일인지 말씀해주세요"
14:45 - 1차 분석 완료
15:00 - 고객에게 중간 보고
         "가맹점 요구사항 분석 중입니다"
16:00 - 긴급 대응 전략 회의 (사내)
17:00 - 고객에게 대응 방안 제시
         "1. 가맹점 대표 5명과 간담회 제안
          2. 제3자 중재 (IBS 변호사 참석)
          3. 정책 수정안 검토"
17:30 - 고객 동의
18:00 - 가맹점 대표들에게 공문 발송 (IBS 명의)
익일 14:00 - 간담회 개최 (변호사 2명 참석)
16:00 - 합의안 도출
→ **고객: "변호사님들이 중재해주셔서 분쟁이 해결됐어요"**
```

**결과**:

- 집단 민원 → 건설적 대화로 전환
- 가맹점주들도 IBS 인지
- 본사-가맹점 관계 개선
- 고객사 추천 의지 200%

## ⚡ **긴급 대응 체계 운영 가이드**

### **1단계: 긴급도 분류 (3분 이내)**

| **등급** | **상황** | **대응 시간** | **담당자** |
| --- | --- | --- | --- |
| 🔴 **최고 긴급** |   • 소송 위협
  • 공정위 조사
  • 언론 위기
  • 집단 민원 | **5분 이내**
변호사 직접 연락 | 대표/수석 변호사
직접 투입 |
| 🟠 **긴급** |   • 계약 긴급 검토
  • 법률 자문 긴급
  • 가맹점 분쟁 | **1시간 이내**
1차 답변 | 담당 변호사
  • 팀장 백업 |
| 🟡 **일반** |   • 일반 문의
  • 정기 자문
  • 서류 검토 | **당일 응답** | 담당 변호사 |

### **2단계: 즉시 대응 프로세스**

**CS 담당자 (접수 후 3분)**:

1. 고객 문의 접수 (전화/문자/카카오톡)
2. 긴급도 판단 (위 기준 참고)
3. Slack 긴급 알림 발송
    
    ```
    @변호사이름 🚨 긴급
    고객: A프랜차이즈
    상황: 가맹점주 소송 위협
    시급도: 최고 긴급
    연락처: 010-XXXX-XXXX
    ```
    
4. 고객에게 즉시 회신
    
    "담당 변호사에게 긴급 전달했습니다. 5분 내 연락드리겠습니다"
    

**담당 변호사 (알림 후 5분)**:

1. 즉시 고객에게 전화
2. 상황 파악 (5~10분 통화)
3. 초기 대응 방향 안내
4. 후속 조치 약속 ("3시간 후 다시 연락드리겠습니다")
5. 사내 보고 (대표 + 팀장)

**팀장/대표 (필요시 즉시)**:

1. 최고 긴급 건은 직접 챙김
2. 리소스 배정 (2명 투입 등)
3. 고객 안심 전화 ("제가 직접 챙기겠습니다")

### **3단계: 중간 보고 (최고 긴급은 2시간마다)**

**중간 보고 예시**:

```
고객님, 담당 변호사입니다.

[현재 상황]
- 계약서 검토 70% 완료
- 핵심 쟁점 3개 발견

[예상 일정]
- 오후 5시: 최종 검토 완료
- 오후 6시: 의견서 전달

궁금하신 점 있으시면 언제든 연락주세요.
```

### **4단계: 사후 관리**

**긴급 대응 완료 후**:

1. 고객 만족도 확인 전화
2. 피드백 수집
3. 사례 문서화
4. 팀 공유 ("이렇게 대응했더니 고객이 감동했어요")
5. 프로세스 개선
