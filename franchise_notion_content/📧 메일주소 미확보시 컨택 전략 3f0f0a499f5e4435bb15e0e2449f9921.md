# 📧 메일주소 미확보시 컨택 전략

## 🎯 문제 정의

**메일주소 없이 어떻게 영업할 것인가?**

- 홈페이지 있어도 대표 메일 미공개: 40%
- 공개된 메일이 일반 고객센터: 60%
- 대표 직통 메일 확보율: 10% 미만

→ **대안 채널 전략 필수**

---

## 📞 채널별 접근 전략

### 우선순위

```
1순위: 전화 (본사 대표번호)
2순위: 카카오톡 비즈니스
3순위: 문자 (SMS/LMS)
4순위: SNS DM (인스타/페이스북)
5순위: 우편 (DM 발송)
6순위: 가맹점 경유
```

---

## 📞 1순위: 전화 컨택

### A. 대표번호 → 대표 연결 전략

**게이트키퍼 통과 스크립트**:

```
[직원] 네, OO프랜차이즈입니다.

[나] 안녕하세요, 
     IBS 법률자문팀 이범휘입니다.
     대표님과 통화 가능할까요?

[직원] 무슨 일이세요?

[나] 프랜차이즈 법률자문 관련해서
     중요한 정보를 전달드리려고 합니다.
     2분만 시간 내주실 수 있을까요?
```

**Tip**: 

- ❌ "영업입니다" (즉시 차단)
- ✅ "중요한 정보" (호기심 자극)
- ✅ 자신감 있는 톤

---

### B. 대표 부재 시 → 메일 주소 확보

```
[직원] 지금 자리에 안 계신데요.

[나] 그럼 메일로 자료 보내드리고 싶은데,
     대표님 메일 주소 알 수 있을까요?

[직원] 저희 고객센터 메일로 보내주세요.

[나] 고객센터 메일로 보내면
     대표님이 직접 보시나요?
     
     아니면 대표님 개인 메일 주소가
     따로 있을까요?
```

**확보 목표**:

- 1차: 대표 개인 메일
- 2차: 비서/총무 메일 ("대표님께 전달 부탁드립니다")
- 3차: 고객센터 메일 ("대표님 참조 부탁드립니다")

---

## 💬 2순위: 카카오톡 비즈니스

### A. 카카오톡 채널 찾기

**검색 방법**:

1. 카카오톡 → 채널 검색 → "회사명"
2. 네이버 검색 → "회사명 카카오톡"
3. 홈페이지 하단 → SNS 아이콘

---

### B. 비즈니스 메시지 전략

**1차 메시지** (관심 유도):

```
안녕하세요 회사명 대표님,
IBS 프랜차이즈 법률자문팀 이범휘입니다.

프랜차이즈 운영 중 
이런 고민 있으신가요?

✅ 가맹계약서 최신 법령 반영 안됨
✅ 정보공개서 업데이트 미루는 중
✅ 가맹점 분쟁 대응 막막함

━━━━━━━━━━━━━━━━━

💡 무료 체크리스트 받아보시겠어요?

[예, 받을게요] [아니요, 괜찮아요]
```

---

**2차 메시지** (응답 없을 때, +3일):

```
대표님, 지난주 메시지 확인하셨나요?

업종 프랜차이즈는
특히 이런 리스크가 많습니다:

⚠️ [업종 특화 리스크 1]
⚠️ [업종 특화 리스크 2]
⚠️ [업종 특화 리스크 3]

━━━━━━━━━━━━━━━━━

작년 OO사는 이걸 몰라서
3억원 손해배상 판결 받았습니다.

우리 회사는 괜찮은지 
체크해보세요. (무료)

[자료 받기]
```

## 📱 3순위: 문자 (SMS/LMS)

### A. 문자 메시지 전략

**1차 문자** (짧고 강렬하게):

```
[IBS 프랜차이즈 법률자문]

회사명 대표님,
가맹계약서 법률 리스크 
무료 점검 받아보세요.

✓ 불공정조항 진단
✓ 정보공개서 체크
✓ 리스크 사전 차단

회신 주시면 즉시 발송
이범휘 02-XXXX-XXXX
```

---

**2차 LMS** (긴 문자, +5일):

```
[IBS] 회사명 대표님께

프랜차이즈 본부가 꼭 알아야 할
법률 리스크 3가지:

━━━━━━━━━━━━━━━━━
1️⃣ 가맹계약서 불공정 조항
   → 계약 무효 + 손배 청구
   
2️⃣ 정보공개서 미등록/허위 기재  
   → 과태료 최대 3,000만원
   
3️⃣ 가맹점 지원 의무 불이행
   → 집단 소송 리스크
━━━━━━━━━━━━━━━━━

실제 사례:
작년 업종 업종 OO사
가맹점 소송으로 3억 배상

━━━━━━━━━━━━━━━━━

무료 체크리스트로 
우리 회사 상태 점검하세요.

회신 또는 전화 주시면
즉시 발송드립니다.

IBS 이범휘
02-XXXX-XXXX
info@ibslaw.co.kr
```

---

### B. 문자 발송 타이밍

**최적 시간**:

- 평일 오전 10-11시 (업무 시작 후)
- 평일 오후 2-3시 (점심 후)

**피해야 할 시간**:

- 출근 시간 (7-9시)
- 점심 시간 (12-1시)
- 퇴근 시간 (6-7시)
- 주말/공휴일

## 📸 4순위: SNS DM

### A. 인스타그램

**계정 찾기**:

- 검색: #회사명, #브랜드명
- 가맹점 게시물 → 본사 태그

**DM 전략**:

```
안녕하세요 회사명 대표님,

IBS 프랜차이즈 법률자문팀입니다.

인스타 통해 연락드려 죄송합니다만,
대표님께 도움이 될 정보가 있어서요.

[이미지 첨부: 체크리스트 썸네일]

프랜차이즈 법률 리스크
무료 체크리스트입니다.

관심 있으시면 
메일이나 카톡 주소 알려주세요!

감사합니다 🙏
```

---

### B. 페이스북

**메신저 전략**:

- 인스타와 동일
- 단, 이미지보다 링크 중심

---

## 📮 5순위: 우편 DM

### A. 우편 발송 전략

**언제 사용?**

- 모든 채널 실패 후 최후 수단
- 고가 타겟 (가맹점 100개 이상)

**발송 내용**:

```
[표지]
회사명 대표님 귀하

[1페이지]
프랜차이즈 법률 리스크
무료 진단 리포트

[2-3페이지]
▸ 업종별 리스크 TOP 3
▸ 실제 분쟁 사례
▸ 예방 체크리스트

[4페이지]
IBS 소개 + 연락처
```

**차별화**:

- 일반 광고가 아닌 "가치 제공"
- QR코드 → 온라인 자료
- 후속 전화: "우편 받으셨나요?"

---

## 🏪 6순위: 가맹점 경유

### A. 가맹점주에게 연락

**프로세스**:

1. 가맹점 방문 (고객으로)
2. 자연스럽게 대화:
    
    ```
    "여기 프랜차이즈 정말 잘 되시네요!
     본사 지원이 잘 되나봐요?
     
     혹시 본사 대표님 연락처 
     알 수 있을까요?
     
     저도 프랜차이즈 창업 준비 중인데
     한번 상담 받고 싶어서요."
    ```
    
3. 명함/전화번호 확보
4. 본사 연락: "OO점 통해서 연락처 알게 됐습니다"

**주의**: 

- 가맹점주에게 부담 주지 않기
- 본사 비판 유도하지 않기

## 💡 핵심 원칙

### 성공의 3가지 법칙

1. **끈기 (Persistence)**
    - 1번 연락에 포기 금지
    - 최소 5-7회 터치
2. **가치 우선 (Value First)**
    - "영업"이 아니라 "도움"
    - 무료 자료/체크리스트 제공
3. **멀티채널 (Multi-Channel)**
    - 1개 채널 실패 시 즉시 다음 채널
    - 모든 채널 동시 공략
