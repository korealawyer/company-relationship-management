# 🏢 홈페이지 없는 기업 공략 전략

## 🎯 현실 인식

**소규모 프랜차이즈의 60%는 홈페이지가 없다**

- 가맹점 10개 미만: 홈페이지 보유율 30%
- 가맹점 10-30개: 홈페이지 보유율 60%
- 가맹점 30개 이상: 홈페이지 보유율 90%

→ **Priority C 타겟도 포기할 수 없다**

---

## 🔍 정보 수집 전략

### 방법 1: 공개 정보 활용

#### A. 공정거래위원회 가맹사업정보제공시스템

**URL**: [https://franchise.ftc.go.kr](https://franchise.ftc.go.kr)

**수집 가능 정보**:

- ✅ 회사명 (법인명)
- ✅ 대표자명
- ✅ 브랜드명
- ✅ 업종
- ✅ 가맹점 수
- ✅ 본사 주소
- ✅ 대표 전화번호
- ✅ 정보공개서 등록 여부

**크롤링 프로세스**:

1. 업종별 검색 (치킨/카페/교육 등)
2. 가맹점 수 10-30개 필터
3. 엑셀 다운로드
4. 홈페이지 URL 확인 → 없으면 Priority C

---

#### B. 사업자등록번호 조회 → 네이버/구글 검색

**프로세스**:

```
1. 공정위 시스템에서 법인명 확보
2. "회사명 + 가맹" 구글 검색
3. 블로그/뉴스/SNS에서 정보 수집:
   - 대표 인터뷰
   - 가맹점 모집 공고
   - 브랜드 소개
4. SNS 계정 찾기 (인스타/페이스북)
5. DM/댓글로 컨택 시도
```

---

### 방법 2: 오프라인 정보 활용

#### A. 가맹점 방문 → 본사 연락처 수집

**프로세스**:

1. 근처 가맹점 방문 (고객으로)
2. 자연스럽게 대화:
    
    ```
    "여기 프랜차이즈인가요? 
     저도 창업 알아보는 중인데
     본사 연락처 알 수 있을까요?"
    ```
    
3. 명함/전단지/포스터에서 본사 번호 확보

**주의**: 가맹점주에게 부담 주지 않기

---

#### B. 프랜차이즈 박람회

**연 2-4회 대형 박람회**:

- 서울 프랜차이즈 박람회 (코엑스)
- 부산/대구 지역 박람회

**수집 가능 정보**:

- 명함 (대표 직통/담당자)
- 브로슈어 (사업 계획)
- 직접 상담 (니즈 파악)

**전략**:

- 부스 방문 → 창업 문의 가장
- "법률 자문 필요하세요?" 자연스럽게 제안
- 명함 교환 → 당일 팔로업 메일

---

## 💡 대체 훅 전략 (홈페이지 없을 때)

### 훅 1: 정보공개서 등록 여부

**메일 제목**: [회사명님] 정보공개서 미등록 시 과태료 최대 3,000만원

**본문 오프닝**:

```
회사명 대표님, 안녕하세요.
IBS 프랜차이즈 법률자문팀입니다.

공정거래위원회 가맹사업 정보를 확인하던 중,
회사명의 정보공개서 등록 여부를 체크하게 되었습니다.

혹시 지금 등록되어 있으신가요?

만약 미등록 상태라면
▸ 과태료 최대 3,000만원
▸ 가맹계약 무효
▸ 손해배상 책임

이런 리스크가 있습니다.

무료 체크리스트로 
현재 상태를 진단해드리겠습니다.
```

---

### 훅 2: 업종별 일반 리스크

**메일 제목**: [회사명님] 업종 프랜차이즈가 꼭 알아야 할 법률 리스크 3가지

**본문 오프닝**:

```
회사명 대표님,

업종 프랜차이즈를 운영하시는 걸로 알고 있습니다.

업종 업종은 특히 이런 법률 리스크가 많습니다:

1️⃣ [업종 특화 리스크 1]
2️⃣ [업종 특화 리스크 2]  
3️⃣ [업종 특화 리스크 3]

실제로 작년에 업종 프랜차이즈 OO사는
[구체적 사례]로 OOO만원 손실을 입었습니다.

무료 체크리스트로 
우리 회사 현황을 점검해보세요.
```

**업종별 리스크 예시**:

**치킨/외식**:

- 식품위생법 위반
- 원산지 표시 의무
- 배달대행 계약 분쟁

**카페**:

- 상표권 분쟁 (유사 브랜드)
- 인테리어 비용 분담
- 로열티 산정 기준 불명확

**교육**:

- 학원법 준수 의무
- 강사 계약 (근로자 vs 프리랜서)
- 환불 규정

---

### 훅 3: 경쟁사 사례 (공포 + 해결책)

**메일 제목**: [회사명님] 같은 업종 OO사, 가맹점 분쟁으로 3억 손실

**본문 오프닝**:

```
회사명 대표님,

최근 업종 업종의 OO 프랜차이즈가
가맹점 집단 소송으로 
3억원 배상 판결을 받았습니다.

문제의 원인은 간단했습니다:
▸ 가맹계약서 불공정 조항
▸ 정보공개서 허위 기재
▸ 본사 지원 의무 불이행

"설마 우리한테?"라고 생각하시겠지만,
가맹사업법은 갈수록 까다로워지고 있습니다.

우리 회사는 괜찮은지,
무료 체크리스트로 확인해보세요.
```

## 📞 전화 컨택 전략 (메일 주소 없을 때)

### 전화 스크립트 (Cold Call)

**1단계: 게이트키퍼 통과 (비서/직원)**

```
안녕하세요,
IBS 법률자문팀 이범휘입니다.

대표님과 잠깐 통화 가능할까요?
프랜차이즈 법률 관련해서 
중요한 정보를 전달드리려고 합니다.
```

**만약 "무슨 일이세요?"라고 물으면**:

```
공정거래위원회 가맹사업 정보를 
검토하던 중에 
대표님께 도움이 될 만한 내용이 있어서요.

2분만 시간 내주실 수 있을까요?
```

---

**2단계: 대표 통화 (30초 오프닝)**

```
회사명 대표님, 안녕하세요.
IBS 프랜차이즈 법률자문팀 이범휘입니다.

갑작스러운 전화 죄송합니다.
지금 1분만 시간 괜찮으실까요?

[대기]

공정위 가맹사업정보를 보다가
회사명을 알게 되었는데요,

업종 프랜차이즈 운영하시면서
이런 고민 있으실 것 같아서 연락드렸습니다:

▸ 가맹계약서 검토
▸ 정보공개서 등록/업데이트
▸ 가맹점 분쟁 대응

저희는 프랜차이즈 법률 전문팀이고,
월 100만원으로 
이런 것들을 전부 해결해드립니다.

혹시 관심 있으실까요?
```

---

### 전화 후 즉시 팔로업

**관심 있는 경우**:

1. 메일 주소 확보
2. 즉시 팔로업 메일 발송 (통화 내용 요약)
3. Calendly 링크 전송

**관심 없는 경우**:

1. "카톡이나 문자로 자료 보내드려도 될까요?"
2. 번호 저장 → 3개월 후 재연락 설정

## 💬 카카오톡/문자 전략

### 카톡 메시지 (비즈니스 채널)

**1차 메시지**:

```
안녕하세요 회사명 대표님,
IBS 프랜차이즈 법률자문팀입니다.

업종 프랜차이즈 운영 중 
이런 고민 있으실까요?

✓ 가맹계약서 검토 필요
✓ 정보공개서 등록/업데이트
✓ 가맹점 분쟁 대응

무료 체크리스트 보내드릴까요?
[예] [아니오]
```

---

### 문자 메시지 (SMS)

**1차 문자**:

```

프랜차이즈 법률 무료 체크리스트

✓ 가맹계약서 점검

✓ 리스크 사전 차단

회신 주시면 즉시 발송드립니다.
IBS 이범휘 02-XXXX-XXXX
```

## 🎯 통합 접근 전략

**1주차**:

- D1: 전화 (Cold Call)
- D3: 문자 발송
- D7: 카톡 발송

**2주차**:

- D10: 재전화 ("지난주 연락드렸는데...")
- D14: 업종별 리스크 메일 (주소 확보 시)

**3주차**:

- D21: 마지막 메시지 "더 이상 연락 안 드리겠습니다"

**3개월 후**: 재시도
