- `DEBUG_GUIDE.md` - 디버깅 가이드

### 코드 분석 문서 (중복)
- `CODE_ANALYSIS_FULL.md` - 전체 코드 분석 (구버전)
- `CODE_ANALYSIS_UPDATED.md` - 업데이트된 코드 분석 (구버전)
