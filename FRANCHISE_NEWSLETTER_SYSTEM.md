# 🏪 프랜차이즈 본사 전용 법무·경영 인텔리전스 서비스
> **버전**: v3.0 (기업 가치 중심) | **최종 업데이트**: 2026-03-10
> **포지셔닝**: 단순 법률 자문사 → **"프랜차이즈 경영 파트너"**
> **핵심 원칙**: AI 자동수집·분석 → 변호사 법적 검증 → 정기 자동 발송 (담당자 추가 작업 없음)

---

## 🔑 왜 이 서비스인가?

> 프랜차이즈 본사가 직면하는 리스크의 **80%는 예측 가능합니다.**
> 법령 변경, 공정위 처분 패턴, 가맹점주 집단 민원 — 모두 데이터에 신호가 먼저 나타납니다.
> 이 서비스는 그 신호를 **자동으로 포착해 귀사 상황에 맞게 해석**하여 매주·매월 정기 보고합니다.

### 서비스를 도입하는 이유

| 기존 방식 | 이 서비스 도입 후 |
|---|---|
| 법령 개정 뒤늦게 파악 → 대응 지연 | 시행 D-90부터 귀사 맞춤 체크리스트 자동 수신 |
| 공정위 처분 뉴스 우연히 발견 | 동종업계 처분 즉시 알림 + 귀사 리스크 자동 비교 |
| 가맹점주 집단 소송 직전에야 감지 | 온라인 커뮤니티·공정위 조정 신청 조기경보 |
| 경쟁사 동향 수작업 조사 | 주요 경쟁 브랜드 5개사 자동 인텔리전스 월간 수신 |
| 연 1회 계약서 검토 (비용 부담) | 판례 기반 리스크 조항 자동 감지 + 갱신 D-90 사전 알림 |

## 📋 핵심 서비스 상세

### ① 법령 변경 모니터링 리포트 *(전 플랜 · 월 1회)*

**귀사가 받는 것:**
- 가맹사업법·개인정보보호법·근로기준법·식품위생법 개정사항 중 **귀사 업종과 가맹점 수에 해당하는 항목만** 필터링
- 시행일 기준 D-90 / D-30 / D-7 대응 체크리스트 자동 생성
- 다음 달 예고 입법 미리보기

**예시 제목:** `[3월 법령 브리핑] 가맹금 반환 규정 개정 시행 D-45 — XX브랜드 300개점 영향`

```
1. 이달의 핵심 변경사항 (3줄 요약)
2. 귀사 관련 조항 체크리스트 (해당 없음 → 자동 제외)
3. 시행일 카운트다운 & 필요 조치
4. 다음 달 예고 법령 미리보기
```

→ [예, 검토 요청] [아니오]

### ⑨ 가맹점주 민원·분쟁 조기경보 *(PRO+ · 주 1회)*

**귀사가 받는 것:**
- 공정위 분쟁조정 신청 건수 이상 증가 자동 감지
- 온라인 커뮤니티(네이버 카페, 블라인드) "본사 불만" 키워드 자동 모니터링
- 민원 집중 지역 자동 매핑 + 법무 대응 초안 즉시 제공

```
⚠️ 민원 조기경보 — 이번 주

🔴 HIGH: 경기 남부권 가맹점주 12명 집단 항의 예고
         (온라인 커뮤니티 급증 + 공정위 조정 신청 접수 감지)

✅ 권장 조치: 해당 지역 담당자 즉시 연락 + 법무 대응 초안 준비
```

> 가맹점주 집단 민원은 **"행정처분 → 보도 → 가맹점주 이탈"** 3단 도미노.
> 조기 감지 시 사전 협상으로 소송 90% 예방 가능.
