# [AI #2 전용] IBS CRM 개인소송 자동화 엔진 연동 프롬프트

당신은 IBS 로펌 CRM 소속 프론트엔드 리드 개발자입니다. 다음 2가지 시스템 연동 관련 코드를 즉시 작성해 주세요. 코드는 그대로 복사-붙여넣기 할 수 있게 주석과 함께 풀 코드로 제공해 주세요.

## 🎯 과제 1: `automationEngine.ts` 기일 자동화 추가
**요구사항**:
- 위치: `src/lib/automationEngine.ts`
- 기업소송과 유사하게 개인소송의 완료되지 않은 기일(D-7, D-3 등)에 대해 알림 로그(`addLog`)를 주입하는 `runPersonalDeadlineAlerts()` 함수를 신규 작성하세요.
- `store`가 아닌 `personalStore`를 불러와서 순회해야 합니다.
- 작성 후 `runAll()` 실행 함수 내부에 해당 함수 호출을 포함시키세요.

## 🎯 과제 2: `mockStore.ts` Supabase 동기화 패턴 적용
**요구사항**: 
- 위치: `src/lib/mockStore.ts`
- 현재 로컬스토리지 기반인 `personalStore`에 기존 기업소송에서 사용하는 `sbSyncLatest` 패턴을 동일하게 적용하세요.
- `loadPersonal()`과 `savePersonal()` 함수를 수정하여 `sbSyncLatest` 및 `sbUpsert` 로직이 동작하도록 코드를 재작성하세요.
- 제외 컬럼(`exclude` 배열)에 'deadlines', 'documents', 'paymentPlan'을 적용해야 합니다.
