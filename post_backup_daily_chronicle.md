* **멀티 모델 서버리스 롤백 기동 조치 (404/500/429/502):**
  - AI 통신 장애 시 뜨던 혼선 유발형 에러 `502(Bad Gateway)` 메세지를 명확한 `500` / `429` 멘트로 치환 조작. 
  - 엔드포인트 `404 Access Error` 발생 회피 대안으로 차세대 모델 대신 **구형 Claude 3.5 Sonnet 을 안정기로 선택, 폴백 루프 투입** (비상 원복)
