# 📧 영업 실행 매뉴얼 | 콜드메일·전화·훅전략 통합 v1.0

프랜차이즈 법률자문 사업의 모든 영업 실행 프로세스를 체계화한 통합 매뉴얼입니다.

## 🎯 이 매뉴얼의 목적

**"한 곳에서 모든 영업 액션을 관리한다"**

- 메일 발송부터 전화 응대, 훅 전략까지 모든 터치포인트 표준화
- 설날 등 시즌 이슈 반영한 실전 일정표 제공
- 소규모 프랜차이즈 특성에 맞춘 맞춤형 접근법
- 논리구조·장표·멘트를 모듈화해서 조합 가능
