## 💰 **수익성 시뮬레이션 (현실 모델)**

### **1년차 예상 매출 구성**

```
Basic (50만원) × 150개사 = 월 7,500만원 (60%)
Standard (100만원) × 100개사 = 월 1억원 (30%)  
Premium (200만원) × 50개사 = 월 1억원 (10%)

→ 월 총 매출: 2억 7,500만원
→ 평균 객단가: 91만원
→ 총 고객사: 300개사 (목표 달성)
```

**vs 단일 가격 100만원 × 300개사 = 월 3억**

- **매출 차이**: -2,500만원 (-8%)
- **실제 원가**: **70% 절감** (50명 무제한 제거)
- **순이익**: **3배 증가** (원가 통제)

## 🎯 **초기 3개월 전략 (레퍼런스 구축)**

### **Phase 1: 무료 진단 + 저가 전환**

**1️⃣ 무료 진단 (2~3월, 리드 확보)**

- **무료 제공**: 개인정보처리방침 분석 + 문제점 5개 리포트
- **목표**: 100개사 무료 진단 → 20개사 유료 전환
- **전환 제안**: "Basic 50만원 첫 달 50% 할인 (25만원)"

**2️⃣ Basic 집중 공략 (4~6월, 레퍼런스)**

- **목표**: Basic 50개사 확보
- **이유**: 빠른 계약 + 낮은 운영 부담 + 후기 축적
- **결과**: 4월부터 Standard/Premium 전환율 급상승

**3️⃣ 업셀링 (7~12월, 수익성 개선)**

- **Basic → Standard**: 20% 전환 목표 (10개사)
- **Standard → Premium**: 10% 전환 목표 (1개사)
- **크로스셀**: 심리상담 별도 판매 (월 +20만원)
