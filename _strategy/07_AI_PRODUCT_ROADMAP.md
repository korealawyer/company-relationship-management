# 🤖 AI 기능 로드맵
*(법무법인 CRM AI 기능 단계별 출시 계획 | GPT → 자율 법무 비서)*

---

## 비전

"2028년, AI가 법무법인 루틴 업무의 80%를 처리한다."
변호사는 전략적 판단과 클라이언트 관계에 집중.
AI는 검색, 초안, 분석, 모니터링을 처리.

이 AI 로드맵은 플랫폼의 **핵심 MOAT**이며,
경쟁사가 흉내낼 수 없는 법무 특화 데이터 누적으로 강화됩니다.

---

## Phase 1: 검색 & 조회 AI (2026 Q1~Q2)

### 1-1. AI 판례 검색 (MVP)
**출시 목표**: 2026 Q2
**기술 스택**: OpenAI GPT-4o + 대법원 오픈API + pgvector
**가격**: Pro 플랜 애드온 +월 100만원

기능 명세:
```
입력: 사건 키워드 or 사건 설명 (자유 텍스트)
처리:
  1. 키워드 추출 → 대법원 API 판례 검색
  2. 검색 결과 → GPT-4o에게 요약 지시
  3. 5개 관련 판례 요약 카드 생성
출력:
  - 사건번호, 법원, 선고일
  - 판시 요약 (3줄 이내)
  - "우리 사건과 유사점: [AI 분석]"
  - 원문 링크
```

**성공 지표**: 변호사 1명당 월 평균 판례 검색 10건+

---

### 1-2. 계약서 위험 조항 탐지
**출시 목표**: 2026 Q2
**기술 스택**: GPT-4o + PDF 텍스트 추출 (LangChain + PyPDF)
**가격**: Growth 플랜 이상 기본 포함

기능 명세:
```
입력: 계약서 PDF 업로드
처리:
  1. PDF → 텍스트 추출
  2. GPT-4o로 위험 조항 분석
     - 일방적 해지 조항
     - 과도한 위약금 조항
     - 불명확한 대금 조건
     - 준거법/관할 불리 조항
  3. 위험도 점수화 (High/Medium/Low)
출력:
  - 위험 조항 하이라이트 (색상 코딩)
  - 각 위험 조항 설명 + 수정 제안
  - 전체 계약 위험 점수 (0~100)
```

## Phase 2: 생성 AI (2026 Q3~Q4)

### 2-1. 계약서 초안 자동 생성
**출시 목표**: 2026 Q4
**기술 스택**: GPT-4o + 법률 계약서 파인튜닝 (Fine-tuning)
**가격**: Standard 플랜 이상 기본 포함

지원 계약서 유형 (MVP):
```
1. 공급/용역 계약서 (가장 수요 높음)
2. 전산 개발 용역 계약서
3. 임대차 계약서 (상가/사무실)
4. 주주간 계약서 (스타트업용)
5. 프랜차이즈 가맹 계약서
```

생성 흐름:
```
1. 계약 유형 선택
2. 핵심 조건 입력 (당사자, 계약 목적, 대금, 기간, 특약)
3. AI가 30초 이내 초안 생성
4. 변호사 인라인 편집 + 조항 추가/삭제
5. 최종 저장 → 전자서명 연동 (1클릭)
```

---

### 2-2. 법무 의견서 초안 생성
**출시 목표**: 2026 Q4
**기술 스택**: GPT-4o + 사건 DB 연동

```
입력:
  - 사건 기본 정보 (CRM 자동 로드)
  - 의뢰인 주요 주장 (텍스트 입력)
  - 상대방 주장 요약
처리:
  → 관련 판례 자동 검색 (Phase 1-1 연동)
  → 의견서 구조 자동 생성:
     I. 사실관계
     II. 법적 쟁점
     III. 관련 법령 및 판례
     IV. 검토 의견
     V. 결론
출력: 편집 가능한 초안 문서
```

## Phase 3: 예측 & 분석 AI (2027 Q1~Q2)

### 3-1. 사건 결과 예측 모델
**출시 목표**: 2027 Q1
**기술 스택**: 내부 사건 DB + 판례 DB → ML 분류 모델
**데이터 요구량**: 로펌 500곳 × 평균 사건 200건 = 100만건+ 학습 데이터

```
입력:
  - 사건 유형 (소송/가처분/중재 등)
  - 청구 내용 및 금액
  - 우리측 증거 강도 (변호사 평가 1~5)
  - 관련 판례 유사도 점수
처리:
  → ML 모델 예측
출력:
  - 승소 확률 (%) + 신뢰 구간
  - "유사 사건 [N]건 중 [K]건 승소 ([%])"
  - 리스크 요인 상위 3개
  - 예상 소요 기간
```

---

### 3-2. 기업 법인 리스크 스코어링
**출시 목표**: 2027 Q2
**기술 스택**: GPT-4o + 공시 데이터 API + 내부 계약 DB

```
리스크 스코어 구성요소 (100점):
  - 현재 소송/분쟁 건수 × 청구금액 (30점)
  - 만료 예정 계약 중요도 (20점)
  - 규제 준수 현황 (20점)
  - 재무 상태 (신용등급, 부채비율) (20점)
  - 담당자 교체/조직 불안정 (10점)

출력: 0~100점 스코어 + 핵심 리스크 5개 + 조치 제안
활용: 월간 법무 리포트에 자동 포함
```

## Phase 4: 자율 법무 비서 (2027 Q4~2028)

### 4-1. 자율 계약 검토 에이전트
**출시 목표**: 2028 Q1
**동작**: 계약서 수신 → AI 자동 검토 → 리포트 → 변호사 확인 → 수정 의견 전달

변호사 시간: 2~3시간 → 20분으로 단축

### 4-2. 법령 변경 모니터링 에이전트
**출시 목표**: 2027 Q4
**동작**: 법제처 API 연동 → 의뢰인 관련 법령 변경 감지 → 자동 영향 분석 → 알림

### 4-3. 자율 법무 보조원 (Full Copilot)
**출시 목표**: 2028 H2
**동작**: 변호사가 질문하면 → 판례 검색 + 의견서 초안 + 관련 사건 목록 + 리스크 분석을 한번에 처리

## AI 개발 원칙

```
1. HUMAN-IN-THE-LOOP: AI 결과 = 초안 only, 법적 판단은 변호사 전결
2. EXPLAINABILITY: AI 분석 근거 항상 표시 (판례 번호, 조항 위치)
3. PRIVACY-FIRST: 의뢰인 데이터 GPT 모델 학습 절대 사용 금지
4. AUDIT TRAIL: AI 생성 결과 + 변호사 수정 내역 100% 보존
5. FAIL SAFE: AI 오류 발생 시 수동 모드 자동 전환 + 변호사 알림
```
