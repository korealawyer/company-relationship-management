# 🤝 파트너십 & 채널 전략
*(회계법인 · 세무사 · 변협 · 리셀러 · M&A 파트너 | 간접 판매 채널 구축)*

---

## 비전

"우리 세일즈팀 1명이 아니라, 파트너 100명이 판매한다."
직접 세일즈의 한계를 파트너 채널로 돌파.
파트너가 영업하고, 우리는 제품과 커미션으로 응답한다.

---

## 1. 전략적 파트너 유형별 플레이북

### Type A: 회계법인 & 세무사 파트너

**왜 최고의 채널인가?**
```
회계사 = 매일 기업 CFO와 소통 → 법인 법무 니즈 자연 발생
세무사 = 법인 설립, 세무 신고 시 법무 필요성 직접 체감
신뢰도 = 기존 재무 파트너의 소개 = 로펌 선정 장벽 제거
```

파트너십 모델:
- **레퍼럴 수수료**: 소개한 법인 수임 성사 시 수임료의 10% 지급 (최초 1년)
- **공동 세미나**: 분기별 기업 법무 + 세무 합동 세미나 개최
- **화이트라벨 리포트**: "세무법인 X가 제공하는 법무 현황 리포트" 공동 브랜딩
- **제공 조건**: 비독점, 비밀유지약정(NDA) 체결 필수

```
타겟 파트너:
P0: 한영, 삼일, 안진, 삼정 등 Big4 회계법인 중소기업 부문
P1: 지역 기반 세무법인 (로펌 인근 5km 내)
P2: 스타트업 전담 세무사 (팁스타운, 을지로 클러스터)
```

---

### Type B: 대한변호사협회 & 각 지방변협 파트너십

**목표**: 협회 공식 인증 획득 → 신뢰도 극대화

접근 전략:
1. 대한변협 법조인공제회에 시스템 시연 → 협회 회원 혜택 프로그램 협약
2. 지방변협 정기 세미나 스폰서십 (연 2회 이상)
3. "협회 認定 법무 자동화 솔루션" 인증 추진
4. 변협 디지털 매거진 광고 + 기고 칼럼 (법무 자동화 트렌드)

기대 효과:
- 협회 공식 채널 = 시장 내 신뢰 100% 확보
- 전국 3만 변호사에게 간접 노출

---

### Type C: 리걸테크 API 파트너

**모델**: 우리 플랫폼을 타 리걸테크 서비스에 API로 공급

```
잠재 파트너:
- 로톡 (사건 수임 후 CRM 자동 연동)
- 사실 (법원 판결문 DB 연동)
- 케임(CAIM) (법무 SI 회사 — 우리 엔진 OEM)
- 법원 전자소송 시스템 연동

수익 모델: API 호출당 과금 or 연간 라이선스
```

---

### Type D: HR & ERP 솔루션 파트너

**모델**: 기업용 ERP(SAP, 영림원, 더존) + 우리 법무 모듈 번들 판매

접근:
- 대기업이 ERP 도입 시 법무 관리 모듈로 함께 제안
- 파트너 ERP사가 영업 → 우리가 구현 → 수익 분배 (70:30)

---

### Type E: 이해충돌 의뢰인 연결 파트너십

**모델**: 우리 로펌에서 이해충돌로 수임 거절한 의뢰인 → 파트너 로펌 연결

참고: `_agents/workflows/conflict_check.md` Step 5

- 의뢰인 동의 후 파트너 로펌 소개
- 파트너 로펌이 수임 성사 시 → 소개 수수료 (수임료의 5~10%)
- 법조윤리 규정 범위 내에서 운영

## 2. 파트너 온보딩 & 관리

### 파트너 등급 체계
| 등급 | 조건 | 혜택 |
|---|---|---|
| **Silver Partner** | 연 소개 1~3건 성사 | 수수료 10%, 파트너 배지 |
| **Gold Partner** | 연 소개 4~10건 성사 | 수수료 12%, 공동 마케팅 지원 |
| **Platinum Partner** | 연 소개 11건+ 성사 | 수수료 15%, CEO 직접 소통 채널 |

### 파트너 관리 자동화
```
TRIGGER: 파트너 소개 계약 성사 확인
ACTION:
  → 파트너 계정에 "소개 성사" 표시
  → 수수료 계산 자동화 → 월말 파트너 정산서 자동 생성
  → 파트너 실적 대시보드 업데이트
```
