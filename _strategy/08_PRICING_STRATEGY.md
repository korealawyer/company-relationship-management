## 1. 가격 아키텍처 (Pricing Architecture)

### SaaS 구독 Tier

| Tier | 대상 | 월 구독료 | 변호사 수 | 핵심 차별점 |
|---|---|---|---|---|
| **Basic** | 1인 개업 / 소형 법무사무소 | **100만원** | 1~4명 | 사건 관리 + 마감일 알림 + 의뢰인 포털 |
| **Pro** | 중형 법무법인 | **300만원** | 5~15명 | Basic + 전자서명 + AI 계약서 초안 + 자동 청구 |
| **Growth** | 대형 법무법인 | **700만원** | 16~50명 | Pro + AI 위험 조항 탐지 + 멀티 오피스 + 우선 지원 |
| **Enterprise** | 대형 로펌 / 기업 법무팀 | **1,500만원+** | 50명+ | Growth + AI 판례 검색 + 리스크 스코어링 + 전담 CSM |

### 기업 법인 자문료 Tier (Track A — 자사 로펌)

| Tier | 기업 규모 | 월 자문료 | 포함 내용 |
|---|---|---|---|
| **Starter** | 매출 100억 미만 | **150만원** | 계약 검토 5건/월, 자문 3회 |
| **Standard** | 100~500억 | **300만원** | 계약 검토 무제한, 이사회 지원 포함 |
| **Premium** | 500~1,000억 | **600만원** | Standard + 소송 대응 포함, 월간 리포트 |
| **Enterprise** | 1,000억+ / 그룹사 | **1,200만원+** | 거의 상시 대응, 계열사 통합 관리 |

## 3. 할인 정책 (Discount Policy)

### 허용 할인
| 조건 | 할인율 | 승인 필요 |
|---|---|---|
| 연간 결제 전환 (월→연) | **10% 할인** | 자동 허용 |
| First 3개월 신규 프로모션 | **20% 할인** | CS 매니저 |
| 비영리/공익 법무단체 | **30% 할인** | CEO 승인 |
| 레퍼럴 성공 1건당 | 1개월 무료 | 자동 허용 |
| 파일럿 종료 즉시 계약 | 온보딩 비용 면제 | CS 매니저 |

### 절대 금지 할인
```
❌ 50% 이상 할인 (브랜드 가치 훼손)
❌ 영구 할인 약속 (계약서에 기재 불가)
❌ 경쟁사 대비 "무조건 싸게" 약속 (최저가 보장 금지)
```

## 4. 업셀 트리거 & 타이밍 플레이북

### 업셀 기회 감지 자동화
```sql
-- 매일 실행: 업셀 기회 로펌 목록 추출
SELECT lf.id, lf.name, lf.current_plan,
  COUNT(u.id) as lawyer_count,
  COUNT(c.id) as monthly_cases,
  SUM(al.count) as monthly_automations
FROM law_firms lf
LEFT JOIN users u ON u.law_firm_id = lf.id AND u.role = 'LAWYER'
LEFT JOIN cases c ON c.law_firm_id = lf.id AND c.created_at >= NOW() - INTERVAL '30 days'
LEFT JOIN (
  SELECT law_firm_id, COUNT(*) as count 
  FROM automation_logs 
  WHERE created_at >= NOW() - INTERVAL '30 days'
  GROUP BY law_firm_id
) al ON al.law_firm_id = lf.id
WHERE lf.status = 'active'
HAVING 
  (lf.current_plan = 'basic' AND COUNT(u.id) >= 4) OR  -- 변호사 수 한계
  (lf.current_plan = 'pro' AND SUM(al.count) >= 80) OR  -- 자동화 포화
  (lf.current_plan = 'growth' AND COUNT(c.id) >= 100)    -- 사건 수 포화
```

### 업셀 시나리오별 스크립트

#### Scenario 1: 변호사 한계 초과 (Basic → Pro)
```
"이번 달 변호사 [N]명이 모두 사용 중인데, 
 Pro 플랜으로 전환하면 무제한 팀원 초대가 가능합니다.
 차이는 월 [200]만원인데, 사무장 1명 채용보다 훨씬 합리적입니다."
```

#### Scenario 2: 자동화 건수 급증 (Pro → Growth)
```
"이번 달 자동화로 처리된 업무가 [N]건입니다.
 Growth 플랜의 AI 계약서 위험 분석이 추가되면,
 계약 검토 시간을 [X]시간 더 절약할 수 있습니다."
```

#### Scenario 3: AI 판례 검색 애드온
```
"지난 달 판례 검색 횟수가 [N]건인데,
 AI 판례 검색 모듈을 추가하면 검색 시간을
 건당 2시간에서 5분으로 줄일 수 있습니다."
```

## 6. 경쟁사 대비 가격 포지셔닝

| 경쟁 옵션 | 비용 | 우리 우위 |
|---|---|---|
| 사무장 1명 채용 | 월 300만+ 인건비 + 퇴직금 + 4대보험 | 우리는 300만원에 퇴직도 없고 야근도 함 |
| 자체 개발 | 초기 1억+ + 유지비 월 100만+ | 우리는 3년치 법무 피드백이 이미 반영 |
| 경쟁 리걸테크 | 월 50~100만원 (기능 제한) | 법무법인 특화 = 변호사법 자동화 내장 |
| 엑셀 + 카카오 | 무료 | 사건 누락 1건 = 수임료 손실 > 연간 구독료 |
