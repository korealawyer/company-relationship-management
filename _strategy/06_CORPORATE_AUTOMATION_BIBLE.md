# ⚡ 기업법인 자동화 바이블
*(기업·법인 의뢰인 법무 자동화 완전 가이드 | 100개 자동화 항목 카탈로그)*

---

## 미션

"기업 법무 담당자가 더 이상 변호사에게 직접 물어보지 않아도 된다."
정기 법무 루틴의 70%를 자동화하여, 변호사는 전략적 판단에만 집중한다.

---

## 자동화 분류 체계

```
[TRIGGER]  → 언제 발생하는가
[ACTION]   → 무엇을 자동으로 처리하는가
[ESCALATE] → 사람이 개입해야 하는 시점
[LOG]      → 어디에 기록하는가
```

## CATEGORY A: 법인 생애주기 자동화

### A-1. 설립 후 첫 체크리스트 자동 발송 (법인 신규 등록 시)
```
TRIGGER: clients.type = 'corporation' 신규 INSERT
ACTION:
  → 법인 설립 후 1개월 체크리스트 이메일 자동 발송
    - 사업자등록 완료 여부
    - 법인계좌 개설 여부
    - 사업자등록증 발급 여부
    - 주주명부 작성 여부
    - 이사회 구성 여부
  → 담당 변호사에게 법인 정보 완성도 알림
LOG: automation_logs (trigger: corporation_setup, type: checklist_sent)
```

### A-2. 등기이사 임기 만료 알림 (60/30/7일 전)
```
TRIGGER: corporate_directors.term_expires_at ≤ NOW() + N일
ACTION:
  D-60: 담당 변호사 이메일 "이사 임기 만료 예정 — 연임/교체 검토 필요"
  D-30: 법인 법무 담당자에게 주주총회 일정 조율 요청
  D-7:  긴급 카카오 알림 + 대표 변호사 에스컬레이션
ESCALATE: D-7까지 조치 없으면 대표 변호사 직접 전화 알림
```

### A-3. 주주총회 일정 자동 등록 (회계연도 종료 + 3개월)
```
TRIGGER: 매년 1월 1일 스케줄러
ACTION:
  → 12월 결산 법인 → 3월 말 정기주주총회 cases 자동 등록
  → 담당 변호사에게 "정기주주총회 준비 시작" 알림
  → 주주총회 준비 체크리스트 자동 발송
    - 이사회 소집 결의 (총회 2주 전)
    - 주주 소집 통지 (총회 2주 전)
    - 의안 준비 (재무제표 승인, 이사 선임 등)
    - 의사록 작성
```

## CATEGORY B: 계약 포트폴리오 자동화

### B-1. 모든 계약 만료 자동 추적
```
TRIGGER: pg_cron 매일 09:00
ACTION:
  → contracts.expires_at 기준 D-90/60/30/7 순차 알림
  → "계약 갱신 검토" 우선순위 To-Do 자동 등록
  → `_agents/workflows/contract_renewal.md` 프로세스 시작
```

### B-2. 자동갱신 조항 관리
```
DATA: contracts.auto_renewal = true, contracts.cancel_deadline_days = N
TRIGGER: auto_renewal = true AND expires_at - cancel_deadline_days ≤ TODAY
ACTION:
  → "자동갱신 취소 기한 임박" 긴급 알림
  → 갱신 여부 확인 요청 (법인 담당자 + 변호사)
  → 미응답 24시간 → 에스컬레이션
```

### B-3. 계약 상대방 법인 소송 이력 모니터링
```
TRIGGER: 분기 1회 자동 실행
ACTION:
  → 계약 상대방 법인명으로 대법원 API 소송 이력 조회
  → 신규 소송 발견 시 담당 변호사 알림
  → "상대방 재정 리스크 증가" 경고 리포트 생성
```

## CATEGORY C: 규제·준법 자동화

### C-1. 개인정보처리방침 연간 검토 알림
```
TRIGGER: 매년 1월 1일 (또는 법인 설립일 기념일)
ACTION:
  → "개인정보처리방침 연간 검토 시점" 알림
  → 개인정보보호위원회 최신 가이드라인 링크 첨부
  → 검토 완료 체크리스트 발송
```

### C-2. 산업안전보건법 연간 점검 알림
```
TRIGGER: 매년 3월 1일
ACTION:
  → 제조업/건설업 의뢰인에게 "안전보건관리 점검" 알림
  → 연간 점검 체크리스트 발송
  → 점검 미완료 시 → 담당 변호사 에스컬레이션
```

### C-3. 공정거래법 정기 컴플라이언스 체크 (프랜차이즈)
```
TRIGGER: 분기 1회 (가맹 본부 의뢰인 전용)
ACTION:
  → 정보공개서 최신화 여부 확인 요청
  → 가맹계약서 표준약관 준수 여부 체크
  → 가맹점주 분쟁 접수 현황 확인
  → 이상 감지 시 → 담당 변호사 즉시 알림
```

## CATEGORY D: 분쟁 예방 자동화

### D-1. 내용증명 수신 즉시 케이스 등록
```
TRIGGER: 기업 법무 담당자가 포털에 내용증명 업로드
ACTION:
  → cases 테이블 신규 사건 자동 생성 (type: 'dispute_prevention')
  → 담당 변호사에게 즉시 알림 (30분 이내 확인 요청)
  → 내용증명 수신일 + 30일 = 응답 기한 자동 설정
  → D-7, D-3, D-0 응답 기한 카카오 알림
```

### D-2. 직원 퇴직 급증 리스크 모니터링
```
TRIGGER: 기업 법무 담당자가 월간 인사 현황 업데이트
ACTION:
  → 동일 시기 퇴직자 3명 이상 시 → 노동 분쟁 리스크 경고
  → 담당 변호사에게 "취업규칙 검토 + 퇴직 협의 지원" 제안
  → 관련 판례 3건 자동 검색 + 전달 (`_agents/ai_product.md`)
```

## CATEGORY E: 보고 & 분석 자동화

### E-1. 기업 법무 비용 ROI 분석 (분기)
```
TRIGGER: 분기 마지막 날 18:00
ACTION:
  → 해당 분기 법무 비용 집계
  → 방어한 소송 잠재 손해액 추산 (변호사 입력 기반)
  → "법무 투자 대비 리스크 예방 효과" 리포트 자동 생성
  → 기업 의사결정자(CFO/CEO)에게 발송 준비
```

### E-2. 기업 법인 리스크 대시보드 업데이트 (주간)
```
TRIGGER: 매주 월요일 09:00
ACTION:
  → 모든 기업 의뢰인 리스크 스코어 재계산
  → High Risk (70점 이상) 법인 → 담당 변호사 주간 회의 안건 자동 등록
  → 이번 주 만료 예정 계약/기한 목록 대시보드 업데이트
```
