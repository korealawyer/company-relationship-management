# 💰 SaaS Sales Playbook — 로펌 대상 SaaS 판매 전략 플레이북
*(타 법무법인에 우리 플랫폼을 판매하기 위한 완전한 세일즈 운영 가이드)*

---

## 1. 제품 포지셔닝

### 핵심 메시지 (Single Message)
> **"사무장 1명 인건비로, 로펌 전체 업무 자동화"**

### 포지셔닝 계층 구조
```
Level 1 (모든 로펌):
  "의뢰인 관리 + 사건 관리 + 전자계약 + 수임료 청구를 하나로"

Level 2 (자동화에 관심 있는 로펌):
  "마감일 누락 0건 보장, 카카오 알림 자동화"

Level 3 (기업 법무 전문 로펌):
  "기업 법인 포트폴리오 관리 + AI 법률 리스크 진단"

Level 4 (대형 로펌 / Enterprise):
  "멀티 테넌트 완전 격리 + 커스텀 워크플로우 + 전담 CSM"
```

## 2. 가격 전략 & 요금제

### 공개 요금표 (랜딩 페이지 게시)

| 플랜 | 대상 | 월 구독료 | 포함 |
|---|---|---|---|
| **Basic** | 변호사 1~5명 | ₩100만 | 사건관리+의뢰인관리+기본알림 |
| **Pro** | 변호사 5~20명 | ₩300만 | Basic + 전자계약 + 청구자동화 + 포털 |
| **Growth** | 변호사 20~50명 | ₩700만 | Pro + AI법률지원 + 분석대시보드 + 온보딩 |
| **Enterprise** | 50명+ / 법인법무팀 | ₩1,500만+ | Growth + 전담CSM + 커스텀 + SLA 99.9% |

### 부가 수익 구조
  - 전자계약 건당: ₩3,000 ~ ₩10,000 (초과분)
  - 고급 자동화 모듈: 월 ₩50만 (애드온)
  - 초기 온보딩 세팅비: ₩100만 ~ ₩500만 (일회성)
  - AI 판례 검색 모듈: 월 ₩20만 (애드온)
  - 데이터 마이그레이션 대행: ₩50만 ~ ₩200만 (일회성)

### 할인 전략
  연간 결제 전환: 10% 할인
  첫 3개월 할인: 20% (단, 연간 계약 조건)
  파일럿 파트너 (레퍼런스 동의): 30% 영구 할인
  대한변협 회원: 20% 영구 할인
  ❌ 50% 이상 할인 절대 금지 (브랜드 훼손)

## 3. 1000억 ARR 달성 로드맵

### 단계별 전략

```
[Phase 1] 2026 Q1~Q2: "레퍼런스 확보" (MRR 목표: 5,000만)
  전략: 지인 로펌 5곳 무료 파일럿 → 레퍼런스 확보
  목표: 파일럿 5곳 → 유료 전환 3곳 이상
  핵심 지표: NPS > 60, 유료 전환율 > 60%

[Phase 2] 2026 Q3~Q4: "채널 확장" (MRR 목표: 2.5억)
  전략: 변협 파트너십 + SEO 1페이지 + 리셀러 3개 확보
  목표: 월 신규 40곳
  핵심 지표: SEO "법무법인 CRM" 3위 내

[Phase 3] 2027: "복리 성장" (MRR 목표: 8억)
  전략: NRR 115% 달성 (업셀 > 이탈)
  목표: 로펌 200곳, Enterprise 5곳
  핵심 지표: Churn < 2%, NRR > 115%

[Phase 4] 2028~2029: "시장 지배" (ARR 1000억)  
  전략: 해외 확장 + 리걸테크 API 판매
  목표: 로펌 1,000곳 + 기업 법무팀 100곳
  핵심 지표: 시장 점유율 30% 이상
```

## 4. 세일즈 모션별 실행 가이드

### 4.1 인바운드 리드 (웹사이트 → 데모 신청)

처리 시간: 신청 후 24시간 내 연락
과정:
  1. 리드 CRM 등록 (출처, 로펌 규모, 니즈)
  2. 디스커버리 콜 예약 (Calendly 자동화)
  3. 디스커버리 콜 → Pain Point 수집 → 맞춤 데모 준비
  4. 데모 → 파일럿 제안
  5. 파일럿 D21 → 유료 전환 제안

### 4.2 아웃바운드 리드 (콜드 이메일/전화)

타겟 리스트 구축:
  - 대한변협 회원 검색 → 로펌 홈페이지 → 연락처 수집
  - 법원 인근 로펌 지도 검색 → 오프라인 방문
  - 법률 세미나 참석자 명단

첫 이메일 템플릿:
  ```
  제목: [로펌명] 마감일 알림, 자동화하고 계신가요?

  안녕하세요, [담당자명] 변호사님.

  [참조: 홈페이지에서 확인한 전문 분야]를 전문으로 하시는 로펌이시죠?

  저는 변호사 5~50명 규모 법무법인의 사건 관리 자동화를 도와드리는
  [플랫폼명]의 [이름]입니다.

  마감일 알림, 의뢰인 포털, 전자계약 자동화로
  현재 저희와 함께하는 로펌의 반복 업무가 평균 40% 줄었습니다.

  10분이라도 시간 내주신다면 실제 사례를 보여드리고 싶습니다.
  이번 주 가능한 시간이 언제이신가요?
  ```

### 4.3 파트너 채널 (리셀러 기인)

처리:
  - 리셀러 → 파트너 포털에서 Trial 계정 직접 생성
  - 우리 Sales팀 → 공동 데모 지원
  - 계약 성사 → 자동 수수료 정산 (다음 달 15일)

## 5. 경쟁 분기 방어 전략

### 경쟁사 침입 대응

| 시나리오 | 우리 대응 전략 |
|---|---|
| 리걸테크 스타트업 유사 기능 출시 | 법무 데이터 3년 누적 MOAT 강조 + 전환 비용 부각 |
| 대형 ERP 로펌 법무 모듈 출시 | "로펌 전용 설계 vs 범용" 차별화 + 사용 편의성 |
| 가격 경쟁 (덤핑) | 자동화 ROI 계산 제시 → 가격보다 가치 강조 |
| 고객사가 직접 개발 검토 | "3년 개발 비용 vs 월 30만원" 비교 + 법무 특화 기능 목록 |

### MOAT 복제 불가 요소
  1. **법무 데이터 누적**: 실제 로펌 사용 데이터 → AI 정확도↑
  2. **법무법인 특화 워크플로우**: 범용 CRM이 담을 수 없는 법무 프로세스
  3. **변호사법/PIPA 내재화**: 법적 컴플라이언스가 제품에 녹아있음
  4. **이탈 비용**: 3년 누적 사건 이력 + 의뢰인 관계 맵 → 재구축 불가
