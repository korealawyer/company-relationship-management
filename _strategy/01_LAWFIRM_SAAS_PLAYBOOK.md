## 1. 법무법인 Pain Point 15가지 & 솔루션 매핑

| # | Pain Point | 현재 해결책 | 우리 솔루션 |
|---|---|---|---|
| 1 | 사건 마감일 놓침 | 수첩/달력 | 카카오 자동 알림 (D-30/14/7/3/0) |
| 2 | 의뢰인 진행상황 문의 폭주 | 개별 전화 | 의뢰인 포털 (24시간 셀프 조회) |
| 3 | 수임 계약서 도장 받으러 왕복 | 직접 방문 | 전자서명 (카카오/이메일, 3분) |
| 4 | 미납 수임료 관리 실패 | 엑셀 + 수동 전화 | 자동 재촉 알림 + 납부 현황 |
| 5 | 변호사별 업무 현황 파악 불가 | 주간 회의 | 실시간 대시보드 |
| 6 | 사건 서류 관리 카오스 | 이메일/카카오방 | 사건별 문서 보관함 |
| 7 | 신규 상담 누락 (전화 많이 올 때) | 수기 메모 | 상담 접수 자동 등록 + 알림 |
| 8 | 퇴직 직원의 업무 인수인계 | 구두 설명 | 전체 이력 시스템에 보존 |
| 9 | 변호사 이해충돌 체크 누락 | 기억에 의존 | 상대방 DB 자동 교차 검사 |
| 10 | 의뢰인 만족도 파악 불가 | 없음 | 자동 만족도 설문 + NPS |
| 11 | 영업 파이프라인 가시성 없음 | 없음 | Kanban 수임 파이프라인 |
| 12 | 청구서 수동 발행 반복 작업 | 한글/엑셀 | 자동 청구서 + 이메일 발송 |
| 13 | 다수 로펌 지점 통합 관리 불가 | 각 지점 개별 | 단일 플랫폼 멀티 오피스 뷰 |
| 14 | 사건 통계/리포트 없음 | 없음 | 월별 수임료/사건 수 자동 리포트 |
| 15 | 재택/출장 중 접근 불가 | VPN/원격 | 클라우드 기반 모바일 지원 |

## 2. 이상적 고객 프로필 (ICP)

### ✅ 우리가 공략할 로펌
```
규모: 변호사 5~20명
특화: 기업법무, 프랜차이즈법, 노동법, 부동산법
상황: 현재 엑셀/카카오방으로 관리 중
인물: 40대 이하 대표 변호사 또는 IT 담당 사무장 존재
예산: 직원 1명 인건비(300만+)에 준하는 IT 투자 의지
```

### ❌ 피해야 할 로펌
```
- 변호사 1~2명 solo 사무소 (예산 없음, 구조 단순)
- 대형 로펌 50명+ (의사결정 6개월+, RFP 프로세스)
- 이미 타 솔루션 계약 후 6개월 미만 (전환 비용 저항)
```

## 3. 세일즈 사이클 타임라인

```
Day 0:  콜드 아웃리치 / 소개 연결
Day 3:  Discovery Call (20분) — Pain Point 확인
Day 7:  맞춤 데모 (40분) — Pain 순서대로 기능 시연
Day 10: 파일럿 제안서 발송 (30일 무료, 온보딩 포함)
Day 14: 파일럿 시작 (온보딩 세션 Zoom 30분)
Day 21: 중간 체크인 콜 + 유료 전환 제안
Day 30: 파일럿 종료 — 유료 계약 또는 해지
Day 45: 목표 계약 완료 (총 세일즈 사이클 45일)
```

## 4. 데모 시나리오 (40분 표준 스크립트)

### 오프닝 (5분)
```
"오늘 30~40분 안에 세 가지를 보여드리겠습니다.
첫째, 지금 [Pain Point A] 문제를 어떻게 해결하는지.
둘째, [Pain Point B] 자동화가 어떻게 작동하는지.
셋째, 연간 절약되는 시간이 얼마인지 같이 계산해볼게요."
```

### 핵심 기능 시연 (25분)
```
1. 사건 등록 → 변호사 배정 → 카카오 알림 (7분)
   "지금 신규 상담이 들어왔다고 가정합니다..."
   
2. 마감일 자동 알림 설정 (3분)
   "설정 한 번으로 D-7, D-3, D-0 카카오가 자동 발송됩니다"
   
3. 의뢰인 포털 (5분)
   "의뢰인에게 이 링크를 보내면, 진행상황을 스스로 확인합니다"
   
4. 전자서명 계약서 (5분)
   "수임계약서를 방문 없이 3분 안에 처리합니다"
   
5. 대시보드 (5분)
   "오늘 처리해야 할 것, 이달 수임 현황이 한눈에 보입니다"
```

### 가격 제안 & 클로징 (10분)
```
"현재 변호사 X명이시니, Pro 플랜이 맞습니다. 월 300만원입니다.
대신 오늘 결정해주시면 30일 무료 파일럿에 저희가 
기존 데이터 마이그레이션까지 도와드립니다.
언제 시작하시겠어요?"
```

## 5. 파일럿 온보딩 체크리스트

```
Day 1 온보딩 세션 (Zoom 30분):
[ ] 관리자 계정 생성 + 권한 설정
[ ] 팀원 계정 초대 (변호사, 사무직원)
[ ] 로펌 정보 입력 (이름, 로고, 연락처)
[ ] 첫 사건 3건 같이 등록 (핸드홀딩)
[ ] 카카오 알림 연동 설정
[ ] 의뢰인 포털 URL 확인

Day 7 체크인:
[ ] 사용 현황 확인 (사건 수, 로그인 횟수)
[ ] 불편한 점 수집 → 즉시 개선 또는 설명
[ ] 추가 교육 필요 여부 확인

Day 21 전환 제안:
[ ] 파일럿 성과 요약 리포트 공유
[ ] 유료 전환 제안 + 연간 결제 할인 제시
[ ] 계약서 발송
```

## 6. 계약 조건 & 해지 정책

```
계약 기간: 월 단위 (언제든 해지) / 연간 계약 (10% 할인)
결제 방법: 신용카드 자동 결제 / 계좌이체 (엔터프라이즈)
해지 조건: 30일 전 서면 통보
환불 정책: 선납 금액 미사용분 환불 (연간 계약 해지 시)
데이터 보관: 해지 후 90일간 데이터 다운로드 가능, 이후 삭제
```

## 7. 경쟁사 포지셔닝

| 경쟁사 | 강점 | 우리 우위 |
|---|---|---|
| 로톡, 법률사무소 플랫폼 | 신규 의뢰인 유입 | 우리는 수임 후 전 과정 — 보완 관계 |
| 일반 CRM (HubSpot, Salesforce) | 브랜드 파워 | 법무법인 전용 특화 — 변호사법/자동화 내장 |
| 엑셀 + 카카오 | 무료 | 사건 누락 1건 = 수임료 손실 > 연간 구독료 |
| 자체 개발 | 완전 커스텀 | 개발 비용 + 유지보수 = 우리 구독료 10년치 |
