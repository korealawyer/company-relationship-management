## 1. 시장 기회 (Total Addressable Market)

### 한국 법무법인 시장

```
전체 법무법인/법률사무소 수: 약 8,000곳 (대한변협 2024 기준)
- 변호사 1~4명 소형: 5,000곳 (62%)
- 변호사 5~20명 중형: 2,400곳 (30%)
- 변호사 20명+ 대형: 600곳 (8%)

ICP (이상적 고객) TAM:
중형 2,400곳 × 월 300만원 = 72억/월 = 864억/년
대형 600곳 × 월 700만원 = 42억/월 = 504억/년
소형 (Basic) 5,000곳 × 월 100만원 = 50억/월 = 600억/년

Korea Total TAM: ~2,000억/년 (구독 수익만)
```

### 글로벌 확장 TAM (Phase 4)
```
아시아 한국계 로펌 (일본/미국/싱가포르/베트남): ~2,000곳
글로벌 중소 법무법인 시장: $15B (글로벌 리걸테크 리포트 2024)
```

## 2. MOAT (경쟁 해자)

### 복제 불가능한 자산

| MOAT 종류 | 구체적 내용 | 강도 |
|---|---|---|
| **데이터 MOAT** | 3년 누적 사건 이력 × 변호사 패턴 × 의뢰인 관계 DB | ★★★★★ |
| **법무 전문성 MOAT** | 변호사법/개인정보법 맞춤 자동화 = 법률가만 설계 가능 | ★★★★☆ |
| **전환 비용 MOAT** | 3년치 사건 데이터 이식 불가 = 해지 비용 극대화 | ★★★★★ |
| **네트워크 효과** | 로펌 수 증가 → 익명 벤치마크 데이터 품질 향상 | ★★★☆☆ |
| **레퍼런스 MOAT** | 우리 로펌이 직접 씀 = "내부 개발 시스템" 신뢰도 | ★★★★☆ |

### 경쟁사가 따라올 수 없는 이유
```
일반 CRM 회사가 법무법인 특화를 만들려면:
  - 변호사법/윤리규정 학습: 1년+
  - 로펌 업무 플로우 이해: 1년+
  - 법무 자동화 신뢰 쌓기: 3년+
  - 데이터 누적: 5년+
→ 우리는 이미 3년 선행 중 (로펌 직접 운영 레퍼런스)
```

## 3. 비즈니스 모델 & 수익 구조

### 2트랙 비즈니스 모델
```
Track A (내부):  자사 로펌 운영 → 레퍼런스 + 제품 개발 비용 절감
Track B (SaaS):  타 로펌 판매 → ARR 1,000억 목표

두 트랙의 Flywheel:
내부 사용 → 기능 완성도 상승 → 외부 판매 신뢰 → 로펌 수 증가
→ 데이터 품질 향상 → AI 정확도 향상 → 이탈률 감소 ♻️
```

### 수익 지표 (현재 기준)
```
Current ARR: [실적 기입]
MRR Growth Rate: [실적 기입]
Gross Margin: ~85% (SaaS 구독, S/W 비용 외 거의 COGS 없음)
NRR: > 110% 목표 (업셀로 기존 고객 ARPU 증가)
Churn Rate: < 2%/월 목표
Payback Period: < 6개월 (LTV/CAC 비율 > 3x)
```

| 역할 | 핵심 강점 | 설명 |
