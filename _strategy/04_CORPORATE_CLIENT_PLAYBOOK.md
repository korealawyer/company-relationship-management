# 🏢 Corporate Client Playbook — 기업·법인 고객 관리 플레이북
*(법인 클라이언트 특화 전략 | CRM 활용법 | 관계 심화 방법론)*

---

이 플레이북은 **기업·법인 의뢰인**을 로펌의 핵심 장기 고객으로 전환하기 위한
완전한 운영 가이드입니다. 우리 CRM SaaS를 타 로펌에 판매할 때의 레퍼런스로도 활용합니다.

---

## 1. 법인 고객이 개인 고객과 다른 이유

| 구분 | 개인 의뢰인 | 기업·법인 의뢰인 |
|---|---|---|
| 사건 수 | 일반적으로 1~3건 | 수십~수백 건/년 |
| 계약 유형 | 단건 수임 | 연간 자문 계약 (월정액) |
| 의사결정자 | 본인 | 법무팀장/CEO |
| LTV (고객 생애 가치) | ₩500만 ~ ₩5,000만 | ₩3,000만 ~ 무제한 |
| 충성도 | 사건 종결 시 이탈 | 관계 지속 시 수년 유지 |
| CRM 활용도 | 낮음 | 매우 높음 (주간 접속) |

## 2. 법인 고객 분류 체계

### Tier A — 핵심 법인 고객 (연간 수임료 1억 이상)
관리 원칙:
  - 전담 파트너 변호사 1명 고정 배정
  - 월 1회 정기 법무 미팅 (이슈 점검 + 관계 강화)
  - 분기 법무현황 리포트 제공
  - 법령 변경 즉시 알림 (72시간 내)
  - VIP 온보딩: 온사이트 방문 세팅

CRM 설정:
  clients.tier = 'A'
  priority = 'high'
  review_cycle = 'monthly'

### Tier B — 성장 법인 고객 (연간 수임료 3,000만 ~ 1억)
관리 원칙:
  - 담당 변호사 배정
  - 분기 1회 정기 미팅
  - 반기 법무현황 리포트
  - Upsell 기회: Tier A 전환 유도

### Tier C — 소형 법인 고객 (연간 수임료 3,000만 미만)
관리 원칙:
  - 필요 시 담당 변호사 배정
  - 자동화 기반 관리 (마감일 알림, 계약 만료 알림)
  - Upsell 기회: 자문 계약 전환 유도

## 3. 법인 고객 수임 파이프라인

```
잠재 기업 발굴
    ↓
법인 정보 수집 (법인등기, 매출 규모, 계열사)
    ↓
법무 니즈 진단 미팅 (30분)
    → "지금 법무팀이 있으신가요?"
    → "저희 자문을 받으시면 [Pain Point]가 해결됩니다"
    ↓
자문 계약 제안 (월 자문료 + 성과보수)
    ↓
수임 계약서 전자서명 → 자동화 시스템 연동
    ↓
정기 법무 지원 시작
    ↓
관계 심화 → Upsell (추가 법인, 계열사 확장)
```

## 4. 기업 법무 자문 계약 구조

### 월간 자문 계약 3단계 제안

```
[Basic 자문] 월 200만원
  - 전화/이메일 법률 상담: 월 5회
  - 일반 계약서 검토: 월 2건
  - 법령 변경 알림

[Standard 자문] 월 500만원
  - 전화/이메일 법률 상담: 무제한
  - 계약서 검토: 월 10건
  - 법인등기 변경 처리: 월 2건
  - 계약서 초안 작성: 월 3건
  - 정기 미팅: 월 1회

[Premium 자문] 월 1,500만원
  - 전화/방문 상담: 무제한
  - 계약서 검토/작성: 무제한
  - 법인등기 전담 처리
  - 소송 대응 (2심까지 포함)
  - 전담 파트너 변호사 직통 유지
  - 분기 이사회 법무 보고
```

## 5. CRM 기업 법인 필드 정의

```sql
-- 기업·법인 의뢰인 확장 필드
clients_corporate {
  client_id uuid FK → clients,

  -- 법인 기본 정보
  corp_registration_number text ENCRYPTED, -- 법인등기번호
  business_registration_number text ENCRYPTED, -- 사업자등록번호
  founding_date date,
  industry text,             -- 업종 코드
  employee_count int,        -- 임직원 수
  annual_revenue_range text, -- 매출 규모 범위

  -- 법인 구조
  parent_company_id uuid FK → clients, -- 모회사
  subsidiaries text[],       -- 자회사 목록
  major_shareholders text,   -- 주요 주주 (JSON)

  -- 법무 담당자
  legal_contact_name text,
  legal_contact_email text,
  legal_contact_phone text,
  ceo_name text,

  -- 계약 관리
  retainer_tier text,        -- A/B/C tier
  retainer_monthly_fee numeric,
  retainer_start_date date,
  retainer_end_date date,
  review_cycle text,         -- monthly/quarterly/biannual

  -- 리스크 관리
  last_risk_assessment_at date,
  risk_score int,            -- 0~10 (높을수록 주의)
  key_legal_risks text[],    -- ['상표 분쟁', '노동 소송 위험']
}
```

## 6. 법인 고객 관계 심화 전략

### 6.1 기념일 자동화
  법인 설립일: 설립기념일 축하 + "법무 연간 리뷰 제안" 이메일
  계약 기념일: "1년간 법무 파트너로 함께해 주셔서..." + 성과 리포트

### 6.2 공유 인사이트 서비스
  업계별 법령 변경 뉴스레터 (월 1회):
    "귀사 업종에 영향을 미치는 최신 법령 변경 사항"
    → 자동화 발송 → 법인 담당자 + 담당 변호사

### 6.3 계열사 확장 (Land & Expand)
  모회사 수임 성공 → 6개월 후:
    "계열사 [자회사명]의 법무도 지원해 드릴 수 있습니다"
  자회사 할인 제안: 계열사 2사 이상 → 20% 그룹 할인
